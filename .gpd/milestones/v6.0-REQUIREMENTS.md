# Requirements: Gap-Closing Route Expansion Beyond v5.0

**Defined:** 2026-03-29
**Core Research Question:** After `v5.0`, which route family gives the best chance of actually shrinking the remaining `149 K` gap: `Hg`-family uplift around the `Hg1223` benchmark, nickelate escalation via pressure, strain, and chemical pressure, or neither?

## Primary Requirements

### Frontier Mapping

- [x] **MAP-01**: Build a frontier headroom map comparing `Hg`-family cuprates, nickelates, and conventional near-ambient controls by best current `Tc`, operating-pressure status, and retention status
- [x] **MAP-02**: Build a control-ledger that identifies the highest-upside uplift levers for each serious route family
- [x] **MAP-03**: Write an explicit negative-control note that screens out routes with too little headroom or too little evidentiary support for post-`v5.0` priority

### Route Expansion Program

- [ ] **DEC-01**: Produce an explicit shortlist of the next route classes and named candidate families worth advancing after `v5.0`
- [ ] **DEC-02**: Produce a next-step route program memo that says what the next milestone should do first
- [ ] **DEC-03**: Keep the `149 K` room-temperature gap explicit and separate benchmark value from consumer readiness in every route memo

### Decision and Validation

- [x] **VALD-01**: Tie every frontier-map claim to current primary-source papers plus the repo’s carried milestone outputs
- [x] **VALD-02**: Reject false progress from pressure-only, onset-only, or theory-only routes when ranking the next path
- [ ] **VALD-03**: End the milestone phase sequence with one primary next route and one secondary route rather than a diffuse watchlist

## Follow-up Requirements

### Extended Work

- **EXTD-01**: Promote the winning route into a concrete next milestone with named candidate materials and explicit observables
- **EXTD-02**: Keep `Hg1223` reproduction support available even if the route-expansion logic elevates nickelates
- **EXTD-03**: Reopen near-ambient conventional routes only if new evidence exceeds the present control benchmark materially

## Out of Scope

| Topic | Reason |
|-------|--------|
| Claiming a room-temperature finished material | Not supported by the present evidence |
| Treating pressure-only or onset-only superconductivity as an ambient practical route | That would inflate progress beyond the evidence |
| Device engineering or industrial scale-up | Premature before any route closes much more of the gap |
| Reopening blind hydride screening | Previous milestones closed that path negatively |

| Requirement | Accuracy Target | Validation Method |
