# Requirements: Hg1223 Protocol Extraction and Reproducibility

**Defined:** 2026-03-29
**Core Research Question:** Given the hydride no-go and the `v3.0` route hierarchy, what exact `Hg1223` pressure-quench protocol, stability window, and control variables define the strongest carried path toward higher ambient-pressure `Tc`, and how should the repo turn that path into a reproducibility program rather than another benchmark slogan?

## Primary Requirements

### Hg1223 Protocol Extraction

- [x] **PROT-01**: Build an exact `Hg1223` protocol ledger from the full benchmark paper, including sample preparation, `P_Q`, `T_Q`, targeted `Tc`, retained `Tc`, and whether each field is directly reported or still missing
- [x] **PROT-02**: Extract the thermal, temporal, structural, and magnetization evidence for the retained `Hg1223` phase, including explicit degradation conditions
- [x] **PROT-03**: Produce an evidence-grade note that separates `Hg1223` fields into confirmed, partially constrained, and still-unreported categories

### PQP Transfer and Control Analysis

- [x] **XFER-01**: Build a transfer table across `Hg1223`, `BST`, and `FeSe` showing `P_Q`, `T_Q`, retained `Tc`, thermal budget, and recovery conditions
- [x] **XFER-02**: Identify which `PQP` controls appear shared across these systems and which remain route-specific
- [x] **XFER-03**: Produce a missing-control ledger that ranks the unreported or underconstrained variables blocking a replication-grade `Hg1223` protocol

### Experiment-Facing Reproducibility Campaign

- [x] **CAMP-01**: Design a minimal but decisive `Hg1223` parameter sweep covering `P_Q`, `T_Q`, `v_Q`, and sample-preparation or oxygen-state variations
- [x] **CAMP-02**: Define a characterization sequence that separates in-DAC confirmation, cryogenic stability testing, retrieval, and ex-DAC measurements
- [x] **CAMP-03**: Define replication, stability, and failure gates for claiming a real `Hg1223` reproducibility basin

### Decision and Validation

- [x] **VALD-01**: Tie every decisive `Hg1223` claim to primary-source fields with explicit dates, conditions, and reported uncertainties or caveats
- [x] **VALD-02**: The milestone must state clearly whether `Hg1223` is closer to an experiment-grade route after the protocol audit, including its remaining blockers
- [x] **VALD-03**: The milestone closeout must preserve the nickelate backup if the `Hg1223` route remains narrow or fragile, while still naming the primary next experiment set

## Follow-up Requirements

### Extended Work

- **EXTD-01**: Convert the `Hg1223` campaign into collaborator-facing experimental guidance
- **EXTD-02**: Revisit the nickelate backup route only after the `Hg1223` control map is explicit
- **EXTD-03**: Add computational support around oxygen, strain, or defect stabilization only after the experimental control variables are fixed

## Out of Scope

| Topic | Reason |
|-------|--------|
| Claiming a room-temperature finished material | Not supported by the present evidence |
| Device or interconnect engineering | Premature before a much stronger ambient route exists |
| Blind high-throughput global search | Outside the scope of this milestone |
| Reopening a hydride-only candidate hunt | The previous milestones already closed that path negatively |
