# Metastability And Quench Route Map

## Purpose

This route map compares the main pathways by which a pressure-supported or metastable superconducting state might survive to ambient pressure. It distinguishes direct hydride evidence from analog systems and framework routes.

## Route Map

## What These Routes Prove

- `RbPH3` and `Mg2IrH6` show that hydride-specific ambient-retention ideas exist, but they remain theoretical.
- `KB3C3`, `KRbB6C6`, and `PbNH4B6C6` show that ambient or ambient-leaning high-Tc design can be more plausible in rigid light-element frameworks than in pure hydrides.
- `Hg1223` proves that pressure-quench is a real route class for retained high `Tc`, but it does **not** prove hydride quenchability.
- `SrB3C3` is useful because it separates sample survival from ambient superconductivity; those are not the same result.

## What These Routes Do Not Prove

- No non-hydride analog proves that a hydrogen-rich phase will survive decompression.
- No theory-only metastable hydride prediction proves retained superconductivity after a real pressure cycle.
- No framework route in this source set justifies skipping the direct decompression question for `CsInH3`.

## Comparator Shortlist For Later Phases

- **Primary hydride comparator:** `RbPH3`
  - Why: closest hydride-side example of a moderate-pressure synthesis route with a direct `0 GPa` superconductivity claim.

- **Secondary hydride comparator:** `Mg2IrH6`
  - Why: highest-upside ambient hydride claim in the literature set, but the disagreement itself is part of the planning signal.

- **Best pressure-quench analog:** `Hg1223`
  - Why: strongest existing evidence that a pressure-treated high-`Tc` state can survive to ambient pressure.

- **Best framework analog:** `KB3C3`
  - Why: combines anharmonic stabilization, ambient operation, and a high-pressure-to-ambient narrative that is closer to the milestone's practical question than a generic ambient superconductor.

- **Best clathrate family for candidate search:** `KRbB6C6` plus `PbNH4B6C6`
  - Why: they set the realistic `~100-115 K` ambient framework ceiling in the present source set and provide Phase 08 search targets if hydride decompression fails.
