# Phase 21 Route Gates

The first instrumented `Hg1223` campaign should update route confidence only through the evidence tiers from Phase `20`.

## Post-Campaign Route Actions

| Route action | Minimum evidence | Meaning |
| --- | --- | --- |
| **keep primary** | at least one valid `T1` headline-reproduction candidate under complete logs | `Hg1223` remains the primary route, but still not basin-proven |
| **hold confidence** | one or more valid but non-decisive `T1` outcomes, or mixed `T1/T0` evidence with no clean failure | the route stays primary for now, but no upgrade is justified |
| **strengthened route** | at least one fixed node reaches `T2`, and selected follow-up reaches `T3` | `Hg1223` is still fragile but now stronger as an experiment-grade route |
| **downgrade** | no low-`TQ` node reaches even a valid headline-reproduction candidate under countable evidence, and the misses classify as clean route-relevant failures rather than invalidated runs | `Hg1223` remains benchmark-important but weakens as the near-term primary path |

## What These Gates Do Not Allow

- `keep primary` does **not** mean room-temperature progress
- `hold confidence` does **not** mean basin evidence
- `strengthened route` does **not** erase the `149 K` room-temperature gap
- `downgrade` does **not** erase the scientific value of the retained `Hg1223` benchmark

## Gate Logic

1. `T0` outcomes cannot strengthen or downgrade the route.
2. `T1` outcomes can keep the route primary but cannot strengthen it.
3. `T2` is the first tier that supports basin language.
4. `T3` is the first tier that supports "strengthened route" language.

## Consumer Guardrail

Even the strongest gate outcome here still leaves the repo without a room-temperature superconductor. The best carried retained `Hg1223` scale remains `151 K`, still `149 K` below room temperature.
