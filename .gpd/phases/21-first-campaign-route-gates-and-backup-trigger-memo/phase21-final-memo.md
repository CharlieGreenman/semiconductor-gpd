# Phase 21 Final Memo

## v5.0 Result

`v5.0` did not find a room-temperature superconductor.

What it did do is tighten the research program around the current best retained-ambient route:

- the first `Hg1223` campaign is now a collaborator-facing Stage `A` runbook
- every countable run now requires explicit `PQ/TQ/vQ`, sample-state, and thermal-path records
- the repo now has a failure-mode map, an evidence ladder, and route gates that stop ambiguous evidence from sounding decisive
- the nickelate backup is preserved with explicit activation triggers instead of vague fallback language

## What Remains Unknown

- whether the carried low-`TQ` benchmark window survives the first instrumented test
- whether `Hg1223` has a real reproducibility basin or only isolated benchmark hits
- how much of the remaining variance is driven by sample state versus quench trajectory

## Room-Temperature Status

The repo still does **not** have a room-temperature superconductor.

The best carried retained benchmark remains `Hg1223` at up to `151 K`, still `149 K` below `300 K`.

## Next Step

The primary next move is now explicit:

1. execute the instrumented Stage `A` `Hg1223` campaign exactly as packaged in Phase `19`
2. classify the outcomes with the Phase `20` failure map and evidence ladder
3. update route confidence only through the Phase `21` route gates

## Backup Contingency

If the first instrumented campaign fails cleanly under countable evidence or remains stuck below `T2`, the next milestone should strengthen the bilayer nickelate route into a co-primary discovery program while keeping `Hg1223` as the retained-benchmark route.

## Bottom Line

The repo is more scientifically useful than before, but not closer to a room-temperature breakthrough in the headline sense. The real gain is sharper experimental decision-making around the strongest carried route.
