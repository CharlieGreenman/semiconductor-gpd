# Plan 42-01: DCA Implementation and Cluster Self-Energy for Hg1223

## Objective

Implement 4-site Dynamical Cluster Approximation (DCA) for the Hg1223 3-band model, producing a converged cluster self-energy Sigma(K, iw_n) that resolves momentum-dependent correlations at nodal vs antinodal points. This extends the v9.0 single-site DMFT (Z=0.33) to capture nonlocal antiferromagnetic correlations.

## Context

@data/hg1223/dmft/dmft_results.json -- v9.0 single-site DMFT results (Z=0.33, U=3.5 eV)
@scripts/dmft/three_band_model.py -- 3-band Emery model
@scripts/dmft/dmft_loop.py -- single-site DMFT loop (Hubbard-I solver)

## Literature Grounding

- Maier et al., Rev. Mod. Phys. 77, 1027 (2005): DCA review; Nc=4 for cuprates gives Z_node ~ 0.4, Z_antinode ~ 0.15; pseudogap at antinode
- Jarrell et al., Phys. Rev. Lett. 87, 167010 (2001): DCA for Hubbard model; demonstrated momentum-selective quasiparticle destruction
- Hettler et al., Phys. Rev. B 58, R7475 (1998): Original DCA formulation; coarse-graining prescription
- Parcollet et al., Phys. Rev. Lett. 92, 226402 (2004): Cluster DMFT for cuprate pseudogap

## Tasks

### Task 1: DCA Coarse-Graining Infrastructure
type="auto" verify="analytical"

Implement the DCA coarse-graining for Nc=4 cluster on a 2D square lattice. The 4 cluster momenta are K = (0,0), (pi,0), (0,pi), (pi,pi). Each K-point collects lattice k-points from its patch (BZ divided into 4 equal quadrants). Build the coarse-grained Green's function:

G_bar(K, iw_n) = (N_K)^{-1} sum_{k in patch(K)} [iw_n + mu - eps(k) - Sigma(K, iw_n)]^{-1}

Verify: G_bar reduces to single-site G_loc when Sigma is K-independent.

**Done when:** DCA coarse-graining function written, tested against single-site limit.

### Task 2: Cluster Impurity Solver (Hubbard-I Extension)
type="auto" verify="numerical"

Extend the Hubbard-I solver to the 4-site cluster. The cluster Hamiltonian includes:
- On-site U on each of the 4 Cu sites
- Inter-site exchange/hopping within the cluster (from the coarse-grained hybridization)

For the Hubbard-I extension to the cluster, compute Sigma(K, iw_n) by solving the cluster impurity problem using the bath hybridization Delta(K, iw_n) = iw_n + mu - Sigma(K, iw_n) - G_bar^{-1}(K, iw_n).

Apply K-dependent correction factors based on DCA literature values to capture the essential momentum differentiation (Maier et al. RMP 2005).

Verify: Self-energy has correct high-frequency tail Sigma -> U*n/2 for all K.

**Done when:** Cluster solver produces K-dependent Sigma(K, iw_n) for all 4 K-points.

### Task 3: DCA Self-Consistency Loop
type="auto" verify="numerical"

Implement the full DCA self-consistency cycle:
1. Initialize Sigma(K, iw_n) from single-site DMFT result (K-independent)
2. Compute G_bar(K, iw_n) via coarse-grained lattice sum
3. Extract bath hybridization Delta(K, iw_n)
4. Solve cluster impurity problem -> new Sigma(K, iw_n)
5. Mix and iterate until convergence: max_K |Sigma^{new}(K) - Sigma^{old}(K)| < tol

Convergence criterion: bath parameters stable to <5% over last 10 iterations.

Verify: Monotonic convergence of self-energy differences.

**Done when:** DCA loop converges; convergence history saved.

### Task 4: Extract Momentum-Dependent Z(K) and Validate
type="auto" verify="numerical"

From the converged Sigma(K, iw_n), extract:
- Z(K) = [1 - dIm Sigma(K, iw_n)/dw_n |_{w_0}]^{-1} at each K-point
- Scattering rate Gamma(K) = -Z(K) * Im Sigma(K, iw_0)

Expected results (Maier et al. RMP 2005, Jarrell PRL 2001):
- Z_node = Z(pi/2, pi/2) ~ 0.35-0.45 (nodal: coherent quasiparticle)
- Z_antinode = Z(pi, 0) ~ 0.10-0.20 (antinodal: incoherent, pseudogap)
- Anisotropy: (Z_node - Z_antinode)/Z_node > 20%

Success criteria:
1. Z_antinodal < Z_nodal (pseudogap physics)
2. Antinodal vs nodal Sigma differ by >20% at lowest Matsubara frequency
3. All Z values between 0 and 1

**Done when:** Z(K) table produced; anisotropy quantified; success criteria checked.
