# Phase 63: AI Surrogate Model Training and Screening

## Objective

Train a gradient-boosting surrogate on all v1.0-v11.0 computed Tc/lambda/omega_log data plus
curated literature values. Screen 1000+ hypothetical hydrogen-containing layered oxide
compositions and rank by predicted Tc with uncertainty estimates.

## Context

- Phase 58 target zone: lambda = 2.5-4, omega_log = 700-1200 K for Tc = 300 K
- v1.0-v11.0 computed data covers cuprates, nickelates, hydrides, and superlattices
- Literature SuperCon database provides ~30k entries for broader coverage
- Forbidden proxy: surrogate hits are screening flags, NOT Tc predictions

## Tasks

### Task 1: Compile training dataset from v1.0-v11.0 + literature
type="auto"
- Gather all (composition, lambda, omega_log, Tc) from project JSON files
- Add curated literature entries (known superconductors with Tc, composition descriptors)
- Build feature vectors: element fractions, H content, d-electron count, layer count,
  structural type, computed lambda/omega_log where available
- Done: training CSV with >= 200 entries and documented provenance

### Task 2: Train gradient-boosting surrogate model
type="auto"  verify="numerical"
- GradientBoostingRegressor (scikit-learn) for Tc prediction
- 80/20 train/test split, stratified by material family
- Hyperparameter search (n_estimators, max_depth, learning_rate)
- Report: train R^2, test R^2, MAE, feature importances
- Backtracking trigger: if test R^2 < 0.3, fall back to physics rule-based screen
- Done: model with test R^2 > 0.5 saved; feature importance plot

### Task 3: Generate 1000+ screening compositions
type="auto"
- Enumerate ABO3-xHx, A3B2O7-xHx, AB2O4-xHx with specified A/B sets and x range
- Add [BO2]n/[LiH]m superlattice variants
- Compute feature vectors for all candidates
- Done: screening set with >= 1000 compositions

### Task 4: Screen and rank candidates
type="auto"  verify="numerical"
- Predict Tc for all screening compositions
- Ensemble variance from 10 boosting models for uncertainty
- Rank by predicted Tc; identify top 50 (Tc > 200 K) and top 10 (Tc > 250 K)
- Cross-check top 10 against Phase 58 target zone (lambda, omega_log)
- Done: ranked list with uncertainty; top-10 flagged for Phase 64 DFT validation

## Verification

- Train R^2 > 0.7, test R^2 > 0.5
- Feature importance physically sensible (H content, omega_log rank high)
- Top candidates have composition features consistent with Phase 58 targets
- No candidate ranked without uncertainty estimate
