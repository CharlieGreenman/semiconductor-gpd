# Phase 59: Hydrogen-Cuprate Hybrid Structure Design

## Objective

Design and stability-screen two hydrogen-cuprate hybrid structures:
1. Hg1223-H: Replace one apical oxygen with hydrogen in Hg1223
2. [CuO2]/[LiH] superlattice: Ruddlesden-Popper with LiH replacing BaO charge reservoir

For each: construct DFT relaxation input, evaluate stability (E_hull, phonon estimates), and assess electronic structure. Use literature-grounded estimates (no HPC).

## Context

- Phase 58 target: lambda=2.5-4.0, omega_log=700-1200 K for Tc=300 K with d-wave mu*=0
- Hg1223 baseline: a=3.845 A, c=15.78 A, P4/mmm, omega_log~400 K
- Key H-mode frequencies: metal hydride M-H stretch 100-185 meV (1160-2150 K)
- @data/hg1223/relaxed_structure.json
- @data/inverse_eliashberg/target_zone.json

## Tasks

### Task 1: Hg1223-H structure construction
type="auto" verify="analytical"
- Replace apical O at z=0.141 (and z=0.859) with H
- Adjust z-coordinate for expected Cu-H bond length (~1.6-1.8 A vs Cu-O ~2.2 A)
- Write structure JSON and QE input template
- Done: Hg1223-H structure file written with correct stoichiometry

### Task 2: Hg1223-H stability screening
type="auto" verify="numerical"
- Estimate formation energy vs competing phases (Hg, BaO, CaO, Cu, CuO, H2)
- Literature-grounded E_hull estimate
- Evaluate phonon stability qualitatively (H-mode frequency estimate)
- Done: E_hull estimate reported with uncertainty

### Task 3: [CuO2]/[LiH] superlattice construction
type="auto" verify="analytical"
- Build Ruddlesden-Popper structure: [CuO2]2/[LiH]1 and [CuO2]2/[LiH]2
- Lattice parameter matching (LiH rock salt a=4.08 A vs CuO2 a~3.85 A)
- Write structure JSONs
- Done: Two superlattice structures written

### Task 4: Superlattice stability screening
type="auto" verify="numerical"
- Estimate E_hull for both superlattice stoichiometries
- Lattice mismatch strain assessment
- Evaluate thermodynamic viability
- Done: E_hull and stability assessment for both

### Task 5: Electronic structure assessment
type="auto" verify="numerical"
- For each stable structure: estimate N(E_F) from literature analogues
- Check: Cu d-band character near Fermi level preserved?
- Check: does doping from H or LiH maintain metallic CuO2 planes?
- Done: N(E_F) estimates and orbital character assessment for all candidates

### Task 6: Candidate gate
type="auto"
- Apply Phase 59 success criteria: E_hull < 50 meV/atom, no soft modes, N(E_F) > 3 states/eV/cell
- Generate candidate report JSON
- Done: Gate verdict for each candidate

## Verification

- Dimensional consistency: all energies in eV/atom, frequencies in cm^-1 and meV, lattice params in Angstroms
- Stoichiometry check: electron count correct for each structure
- Cross-check: Hg1223-H lattice params within 5% of parent Hg1223
