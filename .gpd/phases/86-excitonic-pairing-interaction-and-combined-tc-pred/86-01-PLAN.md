# Phase 86: Excitonic Pairing Interaction and Combined Tc Prediction

## Objective

Compute excitonic pairing interaction V_ex(q,omega) for Phase 85 candidates using
RPA-like approach. Combine with phonon pairing. Test whether excitonic mechanism
can exceed 240 K Eliashberg ceiling.

## Context

- Phase 85: 3 candidates (1T-TiSe2, SmS golden, kappa-BEDT) with lambda_ex = 0.04-0.16
- Eliashberg ceiling: 240 +/- 30 K
- Key question: additive or double-counting with mu*?

## Tasks

### Task 1: Compute alpha^2F_ex for each candidate (type="auto")

Model the excitonic spectral function as a Lorentzian peaked at omega_ex:
  alpha^2F_ex(omega) = lambda_ex * omega_ex / (2*pi) * Gamma / ((omega - omega_ex)^2 + Gamma^2)
with Gamma = 0.1 * omega_ex (finite exciton lifetime).

### Task 2: Solve combined Eliashberg equations (type="auto")

Use McMillan-Allen-Dynes with combined kernel:
  lambda_total = lambda_ph + lambda_ex
  omega_log_eff = exp[ (lambda_ph * ln(omega_ph) + lambda_ex * ln(omega_ex)) / lambda_total ]
  Tc = f1*f2 * omega_log_eff / 1.2 * exp[-1.04*(1+lambda)/(lambda - mu*(1+0.62*lambda))]

Double-counting check: is lambda_ex already partially in mu*?

### Task 3: 300 K verdict for Track C (type="auto")

Compare Tc_combined vs 240 K ceiling and 300 K target.
Produce mechanism table for Phase 89 consolidation.

## Success Criteria

- alpha^2F_ex computed for 3 candidates
- Combined Tc with uncertainty brackets
- 300 K verdict explicit
- Double-counting assessment
