# Plan 34-01: DMFT Setup and Correlated Electronic Structure for Hg1223

## Objective

Construct a 3-band Hubbard model (Cu-dx2y2, O-px, O-py) for Hg1223 from v8.0 DFT band structure, implement a CTQMC impurity solver workflow, and compute converged DMFT self-energy with quasiparticle weight Z = 0.2-0.5 consistent with Mott-proximity physics.

## Context

- @data/hg1223/electronic_summary.json (v8.0 DFT bands)
- @data/hg1223/relaxed_structure.json (crystal structure)
- v8.0 finding: phonon-only Tc = 31 K, spin fluctuations contribute 55-80%
- Literature: U ~ 3.5 eV, J ~ 0.65 eV for cuprates (Hirayama et al. 2018, Weber et al. 2012)
- Expected: Z ~ 0.3 for optimally doped cuprates

## Requirements

- DM-01: 3-band Hubbard model with U/J from constrained RPA or literature
- DM-02: Converged DMFT self-energy with Z = 0.2-0.5, m*/m = 2-5

## Tasks

### Task 1: Three-Band Hubbard Model Construction
type="auto"
verify="numerical"

Construct the 3-band dp Hubbard model for Hg1223:
1. Build tight-binding Hamiltonian H_0(k) for Cu-dx2y2, O-px, O-py orbitals
   - Use hopping parameters from v8.0 electronic_summary.json and published LDA fits
   - t_pd = 1.3 eV (Cu-O nearest-neighbor), t_pp = 0.65 eV (O-O), eps_d - eps_p = 3.5 eV (charge-transfer energy)
2. Generate Wannier90 input files (projections block, disentanglement window)
3. Set Hubbard parameters: U_d = 3.5 eV, J_d = 0.65 eV (literature values for cuprates)
4. Write model parameter file as JSON for downstream use
5. Validate: band structure from tight-binding matches DFT near Fermi level (bandwidth ~ 2.5 eV)

Done criteria: Tight-binding model reproduces 3-band structure with correct orbital character, hopping parameters saved to JSON, Wannier90 input template generated.

### Task 2: DMFT Solver Setup and Self-Energy Computation
type="auto"
verify="numerical"

Implement single-site DMFT loop with CTQMC solver:
1. Generate CTQMC input files for TRIQS/CTHYB (interaction Hamiltonian, bath parameters, MC parameters)
2. Implement DMFT self-consistency loop script (Python/TRIQS-style):
   - Local Green's function G_loc from lattice sum
   - Weiss field G0 from Dyson equation
   - Impurity solver (CTQMC) produces G_imp, Sigma
   - Self-consistency: G_loc(new) from Sigma via lattice Dyson
3. Compute self-energy Sigma(i*omega_n) on Matsubara axis
4. Extract quasiparticle weight Z = (1 - dSigma/domega|_0)^{-1}
5. Compute spectral function A(k,omega) via analytic continuation (MaxEnt or Pade)
6. Validate against published cuprate DMFT results:
   - Z in range 0.2-0.5
   - m*/m = 1/Z in range 2-5
   - Spectral weight transfer to Hubbard bands visible

Done criteria: DMFT workflow scripts complete, converged self-energy data saved, Z and m*/m computed and within expected ranges, spectral function plotted showing Mott-proximity features.

## Verification

1. Tight-binding bandwidth matches DFT (2.5 +/- 0.3 eV)
2. Z = 0.2-0.5 (Mott proximity)
3. m*/m = 2-5 (ARPES-consistent)
4. Spectral function shows Hubbard band features

## Success Criteria

All DM-01 and DM-02 requirements met. Model and solver workflow ready for Phase 35 spin susceptibility extraction.
