# Phase 22 Research: Gap-Closing Frontier Map and Control Ledger

## Phase Decision

Phase `22` should answer a narrow question with high downstream value: after `v5.0`, which surviving route families still have enough `Tc` headroom and enough controllable physics to justify more work?

## Why This Phase Comes Next

- `v5.0` already solved the experiment-packaging problem for the current `Hg1223` route.
- The next unresolved problem is route choice: stay centered on `Hg`-family uplift, promote nickelates more actively, or admit that neither route now looks better than the carried benchmark.
- The answer should come from current primary sources plus carried repo outputs, not from another diffuse review.

## Source-Constrained Consequences

1. `Hg`-family cuprates remain the absolute-`Tc` leaders among experimentally credible non-hydride routes.
2. Nickelates now have the steepest combined ambient-plus-pressure improvement curve and the richest named control surface.
3. Conventional near-ambient routes remain useful controls, but not the main route-expansion winners.
4. Phase `22` should therefore produce:
   - a frontier headroom map
   - a control-knob matrix
   - a negative-control note

## Guardrail

No Phase `22` output can imply that the repo has a room-temperature superconductor. The role of this phase is to identify the next route worth pushing, not to inflate progress.
