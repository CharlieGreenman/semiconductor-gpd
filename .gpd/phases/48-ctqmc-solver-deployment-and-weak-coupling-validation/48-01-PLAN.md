# Phase 48: CTQMC Solver Deployment and Weak-Coupling Validation

## Objective

Deploy a CT-HYB (continuous-time hybridization expansion) quantum Monte Carlo solver
for the Hg1223 DCA Nc=4 cluster, replacing the Hubbard-I approximation used in v10.0.
Validate in weak-coupling limit and quantify the sign problem.

## Context

- v10.0 used Hubbard-I cluster solver: Z_nodal=0.195, Z_antinodal=0.054
- Hubbard-I overestimates correlation effects (atomic-limit approximation)
- CTQMC is exact within statistical error for the impurity problem
- Expected: CTQMC gives WEAKER self-energy at intermediate coupling (U/W ~ 1)

## Tasks

### Task 1: CT-HYB Solver Implementation
type="auto"

Implement CT-HYB solver for the 4-site Hg1223 cluster:
- Hybridization expansion Monte Carlo sampling
- Measurement of Green's function G(K, tau) and self-energy Sigma(K, iw_n)
- Sign problem tracking (average sign vs temperature)
- Convergence diagnostics (binning analysis for error estimation)

Done: Working solver that produces G(K, iw_n) with statistical error bars.

### Task 2: Weak-Coupling Validation (U -> 0)
type="auto" verify="limiting-case"

Verify CTQMC reproduces Hubbard-I in weak-coupling limit:
- Run at U = 0.1 eV (U/W << 1): both solvers should give Z ~ 1
- Run at U = 1.0 eV: compare Z values, should agree within 10%
- Run at physical U = 3.5 eV: quantify the CTQMC vs Hubbard-I deviation

Done: Z_CTQMC matches Z_HubbardI within statistical error at U = 0.1 eV.

### Task 3: Sign Problem Characterization
type="auto" verify="numerical"

Quantify sign problem severity:
- Average sign <sign> as function of T from 500 K down to 100 K
- At each T: 10^5 MC sweeps, measure <sign> and its variance
- Identify minimum usable temperature (where <sign> > 0.05)
- If <sign> < 0.1 at T = 200 K, flag backtracking trigger

Done: Sign problem map with minimum usable T identified.

### Task 4: Physical-T Self-Energy and Z Extraction
type="auto" verify="numerical"

Run CTQMC at physical parameters (T = 290 K, U = 3.5 eV):
- Extract Sigma(K, iw_n) with error bars from binning
- Compute Z_nodal and Z_antinodal
- Compare with Hubbard-I values: Z_nodal=0.195, Z_antinodal=0.054
- Verify self-energy is smooth and monotonic at low Matsubara frequencies

Done: Z_CTQMC values with statistical uncertainties, comparison table with Hubbard-I.

## Success Criteria

1. CT-HYB converges with statistical error < 5% on self-energy at T >= 100 K
2. Weak-coupling Z matches Hubbard-I within error bars at U = 0.1 eV
3. Sign problem severity quantified; minimum usable T identified
4. Self-energy smooth and monotonic at low frequencies
5. All energies in eV, temperatures in K
