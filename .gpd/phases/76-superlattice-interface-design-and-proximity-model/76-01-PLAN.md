# Phase 76: Superlattice Interface Design and Proximity Model

## Objective

Design superlattice heterostructures with d-wave SC layer proximitized to H-phonon layer. Compute McMillan proximity-coupled Tc as function of layer thicknesses and interface transparency.

## Context

- v11.0: Hg1223 Tc ~ 146 K [106, 216] (CTQMC validated)
- v12.0: best H-oxide lambda_ph = 1.27, omega_ph = 852 K
- v13.0: 300 K requires lambda_ph >= 3.0 + d-wave (mu*=0) + omega_log_eff >= 740 K
- Key physics: proximity effect typically REDUCES the stronger layer's Tc

## Tasks

### Task 1: Define superlattice designs and material parameters
type="auto"

Enumerate candidate bilayer designs:
- Design A: [Hg1223 (CuO2 layer)] / [LaBeH8-like H-phonon layer]
- Design B: [YBCO-type (CuO2)] / [MgH2 H-phonon layer]
- Design C: [Nd0.8Sr0.2NiO2 (NiO2)] / [LiH H-phonon layer]

For each design, tabulate: Tc_S (K), Delta_S (meV), N_S(E_F) (states/eV/spin), xi_S (nm), Tc_N or lambda_ph_N, omega_log_N (K), N_N(E_F), xi_N (nm), estimated t_perp (meV).

done: Parameter table with >= 3 designs, all values with sources/estimates

### Task 2: Implement McMillan proximity model
type="auto"

Implement the McMillan proximity equations for S/N bilayer:
- Tc_eff from coupled gap equations
- Standard McMillan result: gap suppression in S layer, gap induction in N layer
- Include interface transparency parameter Gamma (0 to 1)
- Sweep d_S, d_N layer thicknesses

Key equations:
- Delta_S = Delta_S0 * [1 - (d_N * N_N) / (d_S * N_S) * Gamma] (linearized)
- Full self-consistent: coupled BCS gap equations at the interface
- Cooper limit (d << xi): effective parameters average over bilayer
- De Gennes limit (d >> xi): interface perturbation

done: Python script computing Tc_eff(d_S, d_N, Gamma) for all designs

### Task 3: Compute proximity Tc maps
type="auto"

For each superlattice design:
- Sweep d_S from 0.5 to 10 nm, d_N from 0.5 to 10 nm
- Sweep Gamma from 0.01 to 1.0
- Map Tc_eff(d_S, d_N) at optimal Gamma
- Identify optimal thickness ratio d_S/d_N

done: Tc_eff maps for all designs; optimal parameters identified

### Task 4: Assess pairing symmetry in proximitized layer
type="auto"

For each design, determine whether the proximity-induced gap in the N (H-phonon) layer:
- Preserves d-wave symmetry from the S layer -> mu*=0 applies
- Reverts to s-wave due to isotropic phonon coupling -> mu*=0.10 applies
- Mixed symmetry: s+d hybrid

Key physics: d-wave gap has nodes, so proximity coupling through a non-d-wave channel mixes symmetries. The phonon coupling in the N layer is isotropic (s-wave), which competes with the d-wave proximity gap.

done: Symmetry assessment for each design; effective mu* determined

### Task 5: Cooperative proximity scenario
type="auto"

Test the "cooperative proximity" scenario: can the N layer itself become superconducting with the help of proximity + its own phonon coupling?
- Solve coupled Eliashberg-like equations where:
  - S layer: d-wave gap from spin fluctuations
  - N layer: proximity-induced gap + intrinsic s-wave phonon pairing
- Determine if cooperative effect enhances Tc beyond standalone values

done: Cooperative Tc computed; comparison with standalone Tc_S and Tc_N

## Verification

- Dimensional consistency: all energies in eV/meV, T in K, lengths in nm/Angstrom
- Limiting cases: d_N -> 0 recovers Tc_S; d_S -> 0 gives no SC; Gamma -> 0 gives decoupled layers
- McMillan proximity always reduces Tc_S for N layer without intrinsic pairing
- Known result: BCS proximity effect depresses Tc monotonically with d_N/d_S for normal metal
