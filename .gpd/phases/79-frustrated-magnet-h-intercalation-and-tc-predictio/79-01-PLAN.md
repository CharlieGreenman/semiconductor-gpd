# Phase 79: Frustrated-Magnet H-Intercalation and Tc Prediction

## Objective

For the best frustrated-magnet candidates from Phase 78, design H-intercalated structures, compute omega_log_eff, and predict anisotropic Eliashberg Tc. Quantify whether suppressed lambda_sf (from frustration) allows omega_log_eff to rise enough for 300 K.

## Context

Phase 78 found that frustration suppresses BOTH lambda_sf AND d-wave pairing via chi_s(Q). Track C is expected to close negatively. This phase provides the quantitative documentation:
- Even with H-boosted phonons (omega_ph ~ 1000-2000 K), the total coupling lambda_total drops below 3.0 when lambda_sf is suppressed
- The fundamental trade-off: low lambda_sf -> high omega_log_eff BUT low lambda_total

## Anchors

- Phase 78: lambda_sf values and pairing assessment for frustrated magnets
- v12.0: omega_log_eff = 483 K baseline; lambda_ph = 1.27 for H-oxides
- v13.0: 300 K requires lambda_ph >= 3.0 + d-wave (mu*=0) + omega_log_eff >= 740 K

## Tasks

### Task 1: H-intercalated structure design for top candidates (type="auto")
Design H-intercalated structures for the 3 most promising frustrated magnets:
1. Na_xCoO2.H (replace H2O with H in cobaltate -- already has intercalation chemistry)
2. Cd2Re2O7.H (pyrochlore with lowest lambda_sf from Phase 78)
3. Nd2Ir2O7.H (pyrochlore with moderate lambda_sf)
Estimate: DFT-level stability (E_hull), H-site energetics, phonon spectrum.

### Task 2: omega_log_eff computation with suppressed lambda_sf (type="auto", verify="numerical")
Compute omega_log_eff using the combined formula:
omega_log_eff = exp[ (lambda_ph * ln(omega_ph) + lambda_sf * ln(omega_sf)) / (lambda_ph + lambda_sf) ]
For each H-intercalated candidate with lambda_sf from Phase 78 and estimated lambda_ph from H modes.

### Task 3: Anisotropic Eliashberg Tc prediction (type="auto", verify="numerical")
Predict Tc using Allen-Dynes with mu*=0 (d-wave) for candidates with unconventional pairing,
and mu*=0.10 (s-wave) for candidates with conventional pairing.
Key test: does ANY candidate reach Tc >= 300 K?

### Task 4: Track C closure with quantitative gap accounting (type="auto")
Produce the final Track C verdict:
- lambda_total for each candidate
- omega_log_eff for each candidate
- Tc prediction with uncertainties
- Why Track C fails: the lambda_sf/d-wave coupling through chi_s(Q)
- What would need to change for Track C to work (hypothetical)

## Success Criteria
1. H-intercalated structures designed for >= 3 candidates
2. omega_log_eff computed for each
3. Tc predicted for each with 300 K comparison
4. Track C closure documented with quantitative gap
