# Plan 32-01: Candidate Ranking with Stability Gates

## Objective

Multi-axis ranking of all computed candidates with stability gates applied. Produce a single prioritized candidate list for experimental follow-up.

## Tasks

### Task 1: Assemble full candidate table and apply stability gates (type="auto")

From Phase 31 cross_track_summary.json:
- Extract all 7 candidates
- Add estimated full Tc (phonon + spin-fluctuation) using phonon fraction estimates
- Apply E_hull < 50 meV/atom gate
- Apply dynamic stability gate
- Flag candidates that fail either gate

### Task 2: Multi-axis ranking and top-candidate identification (type="auto")

Rank by 3 axes:
1. Predicted Tc (phonon-only lower bound AND estimated full Tc)
2. Thermodynamic stability (E_hull)
3. Synthetic accessibility (existing precursors, known growth methods)

Produce:
- Ranked table with composite score
- Top candidate identification
- Explicit comparison to Hg1223 baseline (151 K)
- Honest statement if no candidate exceeds Hg1223

## Success Criteria

1. All candidates listed with: structure, ambient Tc (mu*=0.10 and 0.13), lambda, omega_log, E_hull, dynamic stability
2. Three-axis ranking applied
3. E_hull < 50 meV/atom gate enforced
4. Top candidate identified with Hg1223 comparison
5. If no candidate exceeds Hg1223: stated clearly with computational ceiling
