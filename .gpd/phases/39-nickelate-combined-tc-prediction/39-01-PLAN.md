# Plan 39-01: Nickelate Combined Tc Prediction

## Objective

Combine phonon lambda_ph (v8.0) with literature-calibrated spin-fluctuation lambda_sf
to predict total Tc for La3Ni2O7 at 0% and -2% strain using modified Allen-Dynes.
Compare with 80 K experimental target and state the 149 K room-temperature gap explicitly.

## Context

- v8.0 phonon data: lambda_ph = 0.58 (0%), 0.72 (-1.2%), 0.92 (-2%); omega_log = 325 K, 313 K, 296 K
- Phase 38 scalar RPA lambda_sf ~ 0.01-0.03 (underestimate by 10-50x)
- Literature-calibrated lambda_sf: 0.5-1.5 (Sakakibara PRL 2024, Qu PRL 2024)
- Strain enhances nesting by ~15% (Phase 38), so lambda_sf(-2%) > lambda_sf(0%)
- mu* = 0.10-0.13 bracket
- Experimental targets: 80 K ambient film, 96 K pressurized crystal, 40 K ambient bulk

## Tasks

### Task 1: Compute combined Tc across lambda_sf grid (auto)

Implement modified Allen-Dynes formula with lambda_total = lambda_ph + lambda_sf.
Scan lambda_sf in [0.3, 0.5, 0.8, 1.0, 1.2, 1.5] for both strain states.
Use omega_log from phonon data. Apply mu* = 0.10 and 0.13.
Compute f1*f2 strong-coupling corrections.
Record all results in JSON.

**Done criteria:** JSON with Tc predictions for all (strain, lambda_sf, mu*) combinations.
**Verify:** Tc increases monotonically with lambda_sf; Tc(lambda_sf=0) matches v8.0 phonon-only.

### Task 2: Strain dependence and experimental comparison (auto)

Apply 15% strain enhancement to lambda_sf (from Phase 38 nesting trend).
Identify which (lambda_sf, strain) combinations reach 80 K.
Compare predictions with experimental range (40-96 K).
Compute the lambda_sf threshold needed to reach 80 K at each strain.
State the 149 K room-temperature gap explicitly.

**Done criteria:** Comparison table; threshold lambda_sf for 80 K; 149 K gap statement.

### Task 3: Write results JSON and summary (auto)

Save all results to data/nickelate/combined_tc_results.json.
Create SUMMARY.md with full analysis.

**Done criteria:** JSON written, SUMMARY.md complete.

1. Total Tc predicted within 50% of experimental range (40-96 K) for physically reasonable lambda_sf
2. Strain effect quantified: compressive strain increases or decreases Tc
3. 149 K room-temperature gap stated explicitly
4. lambda_sf threshold for 80 K identified
