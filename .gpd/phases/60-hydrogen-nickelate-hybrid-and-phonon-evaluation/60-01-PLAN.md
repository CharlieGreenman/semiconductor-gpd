# Phase 60: Hydrogen-Nickelate Hybrid and Phonon Evaluation

## Objective

1. Design H-intercalated La3Ni2O7: insert H into the rocksalt layer between NiO2 bilayers.
2. Evaluate omega_log for ALL stable candidates from Phases 59+60.
3. Determine: does ANY candidate achieve omega_log > 800 K (Phase 58 target zone entry)?

## Context

- Phase 58 target: lambda=2.5-4.0, omega_log=700-1200 K for Tc=300 K
- Phase 59: ALL cuprate-H candidates FAIL (E_hull >> 50 meV/atom, doping problems)

- Key: nickelate rocksalt layer (LaO) has natural interstitial sites for H
