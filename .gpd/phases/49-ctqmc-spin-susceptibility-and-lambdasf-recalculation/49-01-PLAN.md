# Phase 49: CTQMC Spin Susceptibility and lambda_sf Recalculation

## Objective

Recompute the spin-fluctuation coupling constant lambda_sf using CTQMC-level
precision, quantifying the systematic error of the Hubbard-I approximation.
This is THE critical number for v11.0.

## Context

- v10.0 Hubbard-I: lambda_sf_cluster = 2.88 +/- 0.54
- Phase 48 showed: CTQMC gives Z_nodal 32% higher, Z_antinodal 61% higher
- Weaker self-energy -> weaker vertex -> reduced AF enhancement -> smaller lambda_sf
- Expected: lambda_sf_CTQMC ~ 2.0-2.5 (down from 2.88)

## Tasks

### Task 1: CTQMC Spin Susceptibility chi_sf(q, omega)
type="auto"

Compute the spin susceptibility from CTQMC Green's functions:
- chi_0(q, iw) from CTQMC G(K, iw) (bubble diagram)
- RPA: chi_RPA(q, iw) = chi_0(q, iw) / [1 - U_eff * chi_0(q, iw)]
- Track (pi,pi) peak: is it enhanced or reduced vs Hubbard-I?
- Quantify the CTQMC correction to chi at the AF wavevector

Done: chi_sf with CTQMC Green's functions computed.

### Task 2: lambda_sf Recalculation
type="auto" verify="numerical"

Extract lambda_sf from the CTQMC susceptibility:
- Method A (direct): lambda_sf = N(0) * <V_sf(q)>_FS (Fermi surface average)
- Method B (scaling): lambda_sf_CTQMC = lambda_sf_HI * (chi_CTQMC/chi_HI)
- Both methods should agree; use as cross-check
- Report central value and uncertainty

Done: lambda_sf_CTQMC with statistical + systematic uncertainty.

### Task 3: Systematic Error Quantification
type="auto"

Compare CTQMC and Hubbard-I lambda_sf:
- Ratio lambda_sf_CTQMC / lambda_sf_HI
- Sign and magnitude of correction
- Physical explanation of the correction
- Implications for Tc predictions

Done: Hubbard-I systematic error quantified with sign, magnitude, and physics.

## Success Criteria

1. chi_sf(q, omega) computed with controlled statistical error
2. lambda_sf_CTQMC compared with Hubbard-I 2.88: shift quantified
3. If lambda_sf_CTQMC < 2.88, Hubbard-I overestimates AF -> document
4. Analytic continuation method specified with systematic uncertainty
5. Dimensions correct: chi in states/eV, lambda_sf dimensionless
