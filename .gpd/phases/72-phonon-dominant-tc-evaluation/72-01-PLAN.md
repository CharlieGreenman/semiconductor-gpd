# Phase 72: Phonon-Dominant Tc Evaluation and Strategy Comparison

## Objective

Compute full isotropic Eliashberg Tc for top phonon-dominant candidates from Phase 71
at mu*=0.10, and quantitatively resolve the tradeoff between SF-dominant (d-wave, mu*=0)
and phonon-dominant (s-wave, mu*=0.10) strategies.

## Context

Phase 71 results:
- LaBeH8: omega_eff=912 K, lambda=3.30, Tc=241 K (s-wave) / 321 K (d-wave)
- CaB2H8: omega_eff=1034 K, lambda=2.52, Tc=220 K (s-wave) / 296 K (d-wave)
- BaSiH8: omega_eff=884 K, lambda=2.63, Tc=195 K (s-wave) / 262 K (d-wave)
- SrH10: omega_eff=800 K, lambda=2.85, Tc=188 K (s-wave) / 252 K (d-wave)
- La3Ni2O7-H1.0: omega_eff=632 K, lambda=2.80, Tc=147 K (s-wave) / 197 K (d-wave)

v12.0 SF-dominant baseline: Tc = 197 K (mu*=0, lambda=3.5, omega_eff=483 K)

## Tasks

### Task 1: Full isotropic Eliashberg for top 3 candidates
type="auto" verify="numerical"

Solve the isotropic Eliashberg gap equations on the imaginary-axis for LaBeH8, CaB2H8,
and BaSiH8 using model spectral functions. Compare with Allen-Dynes.

### Task 2: mu* sensitivity analysis
type="auto" verify="numerical"

For each candidate, compute Tc across mu* = 0.00, 0.05, 0.08, 0.10, 0.13, 0.15.
Determine the critical mu* at which phonon-dominant beats SF-dominant.

### Task 3: Strategy comparison table
type="auto" verify="analytical"

Quantitative comparison: SF-dominant (d-wave, mu*=0, low omega_eff) vs
phonon-dominant (s-wave, mu*=0.10, high omega_eff). Which has more headroom?
What would close the remaining gap for each?

### Task 4: Hybrid strategy exploration
type="auto" verify="numerical"

What if a material could have BOTH high omega_ph AND d-wave symmetry?
Test: LaBeH8 parameters with mu*=0 (hypothetical d-wave).
Is this physically achievable? Under what conditions?
