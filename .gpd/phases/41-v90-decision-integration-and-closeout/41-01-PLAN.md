# Plan: v9.0 Decision Integration and Closeout

## Objective

Integrate all v9.0 computational Tc predictions into a final ranking, produce an honest closeout assessment, and generate a near-miss analysis (DEC-02 alternative for the "no candidate exceeds 200 K" case) with concrete recommendations for v10.0.

## Context

- Phase 37: Hg1223 DMFT+Eliashberg Tc = 108 K (central), bracket [70, 148] K; DM-04 PASS at -28.2%
- Phase 39: La3Ni2O7 combined Tc = 54 K (central at -2% strain), range [35, 68] K; SF-03 PASS
- Phase 40: Best screened candidate = Hg1223 strained + 15 GPa at 145 K; no candidate exceeds 200 K centrally
- All prior v8.0 results: phonon-only ceiling 26-36 K for oxides

## Requirements Addressed

- **DEC-01**: Ranking of all candidates by Tc, stability, accessibility, confidence
- **DEC-02**: Near-miss analysis (since no candidate exceeds 200 K)
- **VALD-02**: 149 K gap stated explicitly

## Tasks

### Task 1: Consolidated Candidate Ranking Table (type="auto")

Produce a comprehensive ranking of ALL candidates across v8.0 and v9.0 milestones:
- v8.0 phonon-only candidates (Hg1223, Hg1234, Hg1245, La3Ni2O7, hybrids)
- v9.0 Track A baseline (Hg1223 DMFT+Eliashberg)
- v9.0 Track B (La3Ni2O7 combined)
- v9.0 Track C screened modifications (all 7 from Phase 40)

Columns: Rank, Candidate, Method, Tc_central (K), Tc_range (K), lambda_total, E_hull, Stability, Accessibility, Confidence, Notes.

**Done when:** Machine-readable ranking JSON and formatted markdown table produced.

### Task 2: v9.0 Closeout Memo (type="auto")

Honest assessment answering the core question: "Can beyond-Eliashberg methods predict a material that closes the 149 K gap?"

- Answer: NO -- but they REPRODUCE the benchmark within 30%, validating the computational framework
- Computational Tc ceiling: ~145 K (strained + pressurized Hg1223), which matches the experimental frontier
- 149 K gap: UNCHANGED (experimental gap from 151 K to 300 K)
- Pipeline achievement: DMFT+Eliashberg framework validated for cuprates (first time reproducing Hg1223 Tc within 30%)

**Done when:** Closeout memo written with explicit gap statement and pipeline validation assessment.

### Task 3: Near-Miss Analysis (DEC-02 Alternative) (type="auto")

Since no candidate exceeds 200 K, produce a "what would it take?" analysis:
1. Cluster DMFT uplift estimate: single-site underestimates chi(q,omega) by ~20-50%; cluster DMFT could increase lambda_sf by 0.3-0.9, adding ~15-40 K to Tc
2. Anisotropic Eliashberg (d-wave): 10-30% Tc increase from FS anisotropy, adding ~15-40 K
3. Combined effect: cluster DMFT + anisotropic Eliashberg could plausibly push predictions to 170-210 K range
4. Novel materials: omega_sf > 60 meV would break the cuprate ceiling; what materials might have this?
5. Assessment of each extension's feasibility, effort, and expected impact

**Done when:** Near-miss analysis with quantitative estimates for each extension pathway.

### Task 4: v10.0 Recommendations (type="auto")

Based on the near-miss analysis, produce concrete recommendations:
- Priority 1: Cluster DMFT (DCA, 4-site) for Hg1223 -- tighten lambda_sf uncertainty
- Priority 2: Anisotropic Eliashberg with d-wave gap function -- correct systematic underprediction
- Priority 3: Extended materials screening with validated pipeline
- Estimated timeline and resource requirements for each

**Done when:** v10.0 recommendation memo with prioritized extensions.

### Task 5: Final SUMMARY.md and State Updates (type="auto")

Produce SUMMARY.md with all contract coverage, write closeout ranking JSON, and prepare state updates for return to orchestrator.

**Done when:** SUMMARY.md written, JSON artifacts created, state updates prepared.

## Success Criteria

1. All candidates ranked with Tc, stability, accessibility, and confidence
2. 149 K gap stated explicitly in closeout
3. Honest assessment: no candidate exceeds 200 K; pipeline is validated but ceiling is real
4. Near-miss analysis with quantitative uplift estimates for v10.0 extensions
5. v10.0 recommendations prioritized and justified
