# Phase 14 Route Decision Memo

## Primary Route

`Hg1223`-class pressure-quenched multilayer cuprates

## Backup Route

strain plus oxygen-controlled bilayer nickelate films

## Why This Is The Right Closeout

The repo now has a stronger answer than it had at the end of `v2.0`.

- `v2.0` ended with a benchmark pivot
- `v3.0` turns that pivot into a route hierarchy with explicit design rules

The primary route remains `Hg1223`-class cuprates because nothing else in the carried set is as close to room temperature while already operating at ambient pressure after preparation. The backup route remains bilayer nickelate films because no other ambient platform in the carried set offers a comparably rich combination of strain, oxygen, and processing knobs.

## What This Does And Does Not Mean

It means:

- the repo now has a scientifically grounded next route
- the repo has a backup route that is more controllable and discovery-rich
- the repo has descriptor rules that keep failed hydride narratives out of top contention

It does not mean:

- the repo has found room-temperature consumer hardware
- the primary route is manufacturing-ready
- the room-temperature problem is close to solved

## Remaining Gap

The best carried route is still `149 K` below room temperature.

That gap is too large to allow consumer-hardware claims. The scientific value here is different: the repo now knows where the strongest benchmark, strongest knob surface, and clearest negative controls are.

## Scientific Value

The new science unlocked by this milestone is not a new room-temperature material. It is a sharper discovery program:

- pressure-quench routes must be treated as parameterized protocols, not anecdotes
- structural memory and controllability outrank raw low-pressure hype
- discovery-rich ambient platforms and benchmark-strong routes can be different, and both matter

## Recommendation

If the next milestone is built for `highest current confidence`, center it on `Hg1223`-class pressure-quench and protocol extraction.

If it is built for `highest discovery leverage`, center it on bilayer nickelate strain plus oxygen control while keeping `Hg1223` as the benchmark target.

## Final Consumer Guardrail

The repo still does **not** have a room-temperature consumer-hardware solution.
