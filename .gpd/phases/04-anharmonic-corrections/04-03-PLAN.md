       - CsInH3 at 3 GPa: ONLY if quantum stabilization was confirmed (STABILIZED verdict from Plan 02 Task 2). If UNSTABLE or INCONCLUSIVE, skip 3 GPa.
       - KGaH3 at 10 GPa: converged SSCHA dynamical matrices from Plan 02 Task 1

    2. **Load Phase 3 DFPT e-ph matrix elements:**
       - Extract electron-phonon matrix elements Delta^ab(q) computed by ph.x with electron_phonon='simple'
       - CRITICAL: The DFPT q-grid must be commensurate with the SSCHA 2x2x2 supercell q-grid
       - If Phase 3 used a different q-grid: set up and run a dedicated DFPT e-ph calculation on the 2x2x2 q-grid (additional 8-24 hour computation, documented in this task)

    3. **Basis rotation (elph_fc.x or EPIq workflow):**
       - Rotate DFPT e-ph matrix elements from harmonic eigenvector basis to SSCHA eigenvector basis
       - This step is essential: simply substituting SSCHA frequencies into the harmonic alpha^2F is WRONG (per RESEARCH.md anti-pattern)
       - The rotation: g_SSCHA(q,nu) = sum_mu U_SSCHA(nu,mu) * g_harm(q,mu), where U is the transformation between harmonic and SSCHA eigenvectors

    4. **Compute anharmonic alpha^2F(omega):**
       - alpha^2F_anh(omega) = (1/N_F) sum_{q,nu} delta(omega - omega_nu^SSCHA) * gamma_nu^SSCHA / (2*pi*omega_nu^SSCHA)
       - where gamma_nu^SSCHA uses the rotated e-ph vertices and SSCHA frequencies
       - Use Gaussian broadening sigma = 0.2 meV for delta functions
       - Frequency grid: 0 to max(omega_SSCHA) + 20 meV, step 0.1 meV

    5. **Extract anharmonic lambda and omega_log:**
       - lambda_anh = 2 * integral[alpha^2F_anh(omega)/omega d(omega)]
       - omega_log_anh = exp[(2/lambda_anh) * integral (alpha^2F_anh/omega) * ln(omega) d(omega)]
       - Compare with Phase 3 harmonic values

    6. **Validation:**
       - alpha^2F_anh(omega) >= 0 for all omega (positive-definite check)
       - lambda_anh < lambda_harm (expected for H-rich systems; hardening reduces 1/omega^2 weighting)
       - lambda_anh/lambda_harm in range 0.60-0.85 (consistent with H3S 0.70, YH6 0.70)
       - omega_log_anh > omega_log_harm (hardening increases omega_log)
       - Spectral weight conservation: integral alpha^2F_anh(omega) d(omega) should be within 20% of harmonic (total spectral weight approximately conserved, redistributed to higher frequencies)

    7. **Save data and produce comparison figure:**
       - JSON files with alpha^2F, lambda, omega_log for each candidate
       - Figure: harmonic alpha^2F (dashed) vs anharmonic alpha^2F (solid) for each candidate
       - Inset: bar chart of lambda_harm vs lambda_anh

    Per RESEARCH.md convention: alpha^2F normalization uses lambda = 2 * integral[alpha^2F/omega]. Do NOT use a convention without the factor of 2.
