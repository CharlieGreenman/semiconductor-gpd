  <action>
    Determine whether SSCHA quantum effects stabilize CsInH3 at 3 GPa, where harmonic phonons show marginal instability (min_freq = -3.6 cm^-1).

    This is a TARGETED stability assessment, not a full anharmonic Tc calculation. The goal is a definitive YES/NO on quantum stabilization.

    1. **Enhanced sampling setup:** Because the harmonic imaginary frequency (-3.6 cm^-1) is small and close to the SSCHA noise floor, use ENHANCED sampling:
       - N_configs = 200 per population (2x the standard)
       - Maximum 20 populations
       - Convergence: gradient < 1e-8 Ry^2 AND frequency error bars resolved
       - Per RESEARCH.md Pitfall 4: stochastic error ~ 1/sqrt(N_configs), so 200 configs gives ~0.7x the error of 100 configs

    2. **Load Phase 3 structure:** CsInH3 Pm-3m at 3 GPa (vc-relax output). Create 2x2x2 supercell (40 atoms).

    3. **Load harmonic dynamical matrices at 3 GPa:** These contain the imaginary mode (min_freq = -3.6 cm^-1). The SSCHA will start from these matrices.

    4. **Run SSCHA minimization:** Same workflow as Plan 01 but with 200 configs/pop. Monitor closely:
       - Track the previously-imaginary mode specifically
       - If it becomes real: quantum stabilization is occurring
       - If it stays imaginary: structure is genuinely unstable

    5. **Error bar estimation:** After convergence, run 3-5 additional populations at fixed dynamical matrix to estimate stochastic error bars on phonon frequencies. For the critical mode:
       - If omega_min > 0 AND omega_min - error_bar > 0: STABILIZED (definitive)
       - If omega_min > 0 BUT omega_min - error_bar < 0: MARGINAL (inconclusive)
       - If omega_min < 0: UNSTABLE (definitive)

    6. **Produce verdict figure:**
       - Harmonic vs SSCHA phonon dispersion at 3 GPa
       - Zoom on the region near the imaginary mode
       - Error bars on SSCHA frequencies near the formerly-imaginary mode
       - Color-code: green = stabilized, red = unstable, yellow = inconclusive

    7. **Document implications:**
       - If STABILIZED: CsInH3 at 3 GPa is a valid candidate. Tc(3 GPa) with SSCHA will be computed in Plan 03. This is the highest-Tc regime.
       - If UNSTABLE: CsInH3 3 GPa is eliminated. Tc ceiling for CsInH3 drops to 5 GPa (harmonic Tc = 285 K, SSCHA-corrected ~200-240 K).
       - If INCONCLUSIVE: Report with uncertainty. Recommend 3x3x2 supercell or 500-config populations in future work.

    CRITICAL per RESEARCH.md: Do NOT claim stabilization if error bars overlap zero. Do NOT compute Tc for an unstable structure (fp-unstable-tc).

    Per RESEARCH.md Pitfall 4: This assessment needs at least 200 configs to resolve a -3.6 cm^-1 imaginary mode. With standard 100 configs, the noise would be comparable to the imaginary frequency magnitude.
  </action>
