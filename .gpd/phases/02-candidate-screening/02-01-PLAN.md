<objective>
Construct the computational infrastructure for convex hull screening and compile the competing phase database for 3 ternary hydride systems at 4 pressures.

Purpose: Provide the thermodynamic foundation for candidate screening. Without complete convex hulls including ALL competing binary and elemental phases at each pressure, E_hull values are unreliable (Pitfall 1 from RESEARCH.md). This plan builds the reusable infrastructure and populates the competing phase data.

Output:
- Hull construction code (pymatgen-based, handles ternary A-B-H systems at arbitrary P)
- Structure generators for perovskite (Pm-3m), octahedral (Fm-3m), and clathrate prototypes
- Competing phase enthalpies for K-Ga-H, Rb-In-H, Cs-In-H, Sr-N-B-C-H, Pb-N-B-C-H, and Mg-Ir-H systems at P = 0, 5, 10, 50 GPa
- QE input file templates for vc-relax at target pressures
</objective>

    Create the screening infrastructure:

    1. **screening/structure_generators.py**: Functions to generate candidate crystal structures from prototypes.
       - `perovskite_pm3m(A, B, a_lat)`: Generate Pm-3m ABH3 perovskite. A at corners (0,0,0), B at body center (0.5,0.5,0.5), H at face centers (0.5,0.5,0), (0.5,0,0.5), (0,0.5,0.5). Return ASE Atoms object.
       - `octahedral_fm3m(A, B, a_lat)`: Generate Fm-3m A2BH6 structure. A at 8c Wyckoff, B at 4a, H at 24e. Return ASE Atoms.
       - `clathrate_sodalite(M, a_lat)`: Generate sodalite-type MNH4B6C6 clathrate structure from Wang et al. prototype coordinates.
       - Each function should accept an optional `pressure_GPa` parameter for documentation/metadata purposes.
       - Include initial lattice parameter estimates from literature (Du et al. for perovskites, Wang et al. for clathrates, Lucrezi et al. for octahedral).

    2. **screening/qe_templates.py**: QE input file generators.
       - `generate_vcrelax_input(structure, pressure_GPa, ecutwfc, kgrid, outdir)`: Generate pw.x vc-relax input file. Use Phase 1 validated parameters: ecutwfc=80-100 Ry, conv_thr=1.0d-8, press_conv_thr=0.5 kbar, forc_conv_thr=1.0d-4. Pressure in kbar (multiply GPa by 10). PBEsol functional. ONCV PseudoDojo pseudopotentials.
       - `generate_scf_input(structure, pressure_GPa, ecutwfc, kgrid)`: For fixed-structure SCF after relaxation.
       - `generate_phonon_input(prefix, qgrid, nq)`: ph.x input for DFPT phonons on specified q-grid.
       - Include ASSERT_CONVENTION comment at top of every generated input: "# CONVENTION: PBEsol, ONCV PseudoDojo, pressure in kbar = GPa*10"

    3. **screening/hull_infrastructure.py**: Convex hull construction pipeline.
       - `load_qe_enthalpy(outdir)`: Parse pw.x output, extract total energy E and pressure P. Compute H = E + PV. Convert to eV/atom.
       - `compute_formation_enthalpy(candidate_H, elemental_H_dict, composition)`: Compute Delta_Hf per atom. Use molecular H2 reference (NOT atomic H) for hydrogen.
       - `build_ternary_hull(entries, system_name)`: Build pymatgen PhaseDiagram from list of ComputedEntry objects. Return hull and E_hull for each entry.
       - `validate_hull(hull, known_stable, known_unstable)`: Cross-check hull against known results (e.g., Mg2CoH6 should be near hull; Mg2IrH6 should be ~172 meV/atom above).
       - `plot_hull(hull, candidates, filename)`: Ternary phase diagram with candidates marked.
       - `check_hull_completeness(hull, system)`: Verify all expected binary phases are included. Warn if fewer than 3 stoichiometries per binary subsystem.

    4. **screening/competing_phases.py**: Competing phase database management.
       - `query_mp_binaries(system, api_key=None)`: Query Materials Project for all known binary phases in a ternary system at 0 GPa. Return list of compositions, structures, formation enthalpies.
       - `get_elemental_references(elements, pressure_GPa)`: Return standard state structures for each element at given pressure. For H: molecular H2 in 15-A cubic box. For metals: known ground-state crystal structure.
       - `list_required_competing_phases(system)`: For a ternary A-B-H, enumerate: A, B, H2, AH, AH2, AH3, BH, BH2, BH3, AB, AB2, A2B, and any known ternary phases.
       - Offline mode: if mp-api key unavailable, use hardcoded formation enthalpies from NIST/literature for the specific systems we need.

    Use Phase 1 parameters throughout: ecutwfc=80-100 Ry depending on elements, k-grid 16^3 for cubic cells (scale by inverse lattice parameter for non-cubic), degaussw=0.075 eV.
