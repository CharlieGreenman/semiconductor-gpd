<objective>
Screen the MXH3 perovskite hydride family (KGaH3, RbInH3, CsInH3) for thermodynamic and dynamic stability at P = 0, 5, 10, 50 GPa.

Purpose: The MXH3 perovskites are the highest-priority candidates: Du et al. (2024) predict Tc = 130-153 K at <= 10 GPa for these compounds, with thermodynamic stability. This plan determines whether our independent DFT calculations confirm their stability and advances them to Phase 3 Eliashberg calculations.

Output:
- Formation enthalpies and E_hull at 4 pressures for all 3 candidates
- Convex hull phase diagrams for K-Ga-H, Rb-In-H, Cs-In-H
- Phonon dispersions for all near-hull candidates
- Stability verdicts (pass/fail) for each candidate at each pressure
- Ranked candidate list for Phase 3
</objective>

  <action>
    For each perovskite candidate (KGaH3, RbInH3, CsInH3) at each pressure (0, 5, 10, 50 GPa):

    1. **Generate and relax structures.** Use screening/structure_generators.py to create Pm-3m perovskite structures with initial lattice parameters from Du et al. Run QE vc-relax at each pressure using Phase 1 parameters (ecutwfc=80 Ry for K/Ga/H, 100 Ry for Rb/In/Cs due to heavier elements; k-grid 16^3; PBEsol; ONCV PseudoDojo).
       - Initial lattice parameters (from literature): KGaH3 ~4.0 A, RbInH3 ~4.5 A, CsInH3 ~4.7 A at 0 GPa.
       - Allow full cell relaxation including cell shape (ibrav=0 with CELL_PARAMETERS).

    2. **Compute competing phase enthalpies at each pressure.** Using the database from Plan 02-01:
       - Load 0 GPa enthalpies from Materials Project / literature.
       - For 5, 10, 50 GPa: run vc-relax for each competing phase (or use Murnaghan EOS fit if multiple P-V data points are available).
       - For H2: compute enthalpy of molecular H2 in 15-A box at each P. At 0 GPa: E(H2) is the reference. At finite P: H2 bond compresses; use vc-relax with fixed molecular orientation.

    3. **Construct convex hull at each pressure.** Use screening/hull_infrastructure.py:
       - Build pymatgen PhaseDiagram for K-Ga-H, Rb-In-H, Cs-In-H at each P.
       - Compute E_hull for each candidate.
       - CRITICAL: Verify hull completeness by checking that all binary hydrides (KH, GaH3, RbH, InH3, CsH, InH3) and intermetallics (KGa, RbIn, CsIn) are included. If any are missing, the hull is unreliable.

    4. **Record and report.** Save to data/candidates/perovskite_results.json:
       ```json
       {
         "KGaH3": {
           "0GPa": {"E_hull_meV_atom": X, "Delta_Hf_meV_atom": Y, "a_lat_A": Z, "volume_A3": V},
           "5GPa": {...}, "10GPa": {...}, "50GPa": {...}
         },
         ...
       }
       ```

    5. **Generate hull figures.** One ternary phase diagram per system per pressure. Mark candidate position. Color by E_hull distance.

    **APPLY fp-above-hull FILTER:** Only candidates with E_hull < 50 meV/atom at ANY pressure <= 10 GPa proceed to phonon stability check (Task 2). Candidates stable only at 50 GPa are noted but do NOT advance at this stage.

    **Enthalpy convergence check:** For at least one candidate (KGaH3 at 10 GPa), compare formation enthalpy at ecutwfc=80 vs 100 Ry. Must converge to < 5 meV/atom.

    **PBE cross-check for KGaH3 at 10 GPa:** Du et al. used PBE+PAW; we use PBEsol+ONCV. To assess functional sensitivity, re-relax KGaH3 at 10 GPa with PBE (instead of PBEsol) and recompute formation enthalpy and E_hull. Report:
    - Lattice parameter difference (PBE vs PBEsol): expect PBE gives ~1% larger a.
    - E_hull difference: if > 20 meV/atom, the stability verdict is functional-dependent; flag this.
    - Formation enthalpy difference: document but do not change the primary PBEsol result.
    This cross-check uses the SAME competing phases re-relaxed with PBE, not mixed PBE/PBEsol enthalpies.

<task type="auto">
  <name>Task 2: Phonon stability screening for near-hull perovskite candidates</name>
  <files>data/candidates/perovskite_phonons.json, figures/phonon_KGaH3.pdf, figures/phonon_RbInH3.pdf, figures/phonon_CsInH3.pdf</files>
  <action>
    For ALL candidates that passed the E_hull < 50 meV/atom filter in Task 1 at any pressure <= 10 GPa:

    1. **Compute phonon dispersions via DFPT.** At the pressure where E_hull is minimized (or closest to hull):
       - Run QE ph.x on 4x4x4 q-grid for the relaxed structure.
       - Use ASR='crystal' in q2r.x.
       - Interpolate to dense q-path along Gamma-X-M-Gamma-R high-symmetry path for cubic Pm-3m.
       - Plot phonon dispersion with matdyn.x output.

    2. **q-grid convergence test.** If any imaginary modes are found at 4x4x4:
       - Recompute on 6x6x6 q-grid.
       - Compare: if imaginary modes vanish or shift by > 50 cm^-1, they are convergence artifacts.
       - If imaginary modes persist (same q-point, similar magnitude), they are real instabilities.
       - **8x8x8 convergence step (conditional):** If 4x4x4 and 6x6x6 minimum frequencies disagree by > 5 cm^-1 (for ANY mode, not just imaginary), proceed to 8x8x8. This is mandatory per ROADMAP convergence protocol. If 4x4x4 and 6x6x6 agree to < 5 cm^-1, 8x8x8 may be skipped.
       - Record: "persistent_imaginary: true/false", "max_imaginary_cm-1" value, and "qgrid_converged_at: 6x6x6|8x8x8".

    3. **Mode character analysis for persistent imaginary modes.** If any candidate has persistent imaginary modes:
       - Extract the eigenvector of the imaginary mode from matdyn.x output.
       - Determine: is the displacement primarily H-atom motion (cage-breathing/rattling) or framework distortion (A/B sublattice)?
       - H-dominated imaginary modes are candidates for SSCHA quantum stabilization (note for Phase 4).
       - Framework-dominated imaginary modes indicate a real structural instability at this pressure -> candidate fails dynamic stability at this P.

    4. **Zero-point energy (ZPE) estimation.** For each dynamically stable candidate (all frequencies real after q-grid convergence):
       - Compute ZPE = (1/2) * sum_q,nu hbar*omega(q,nu) from the converged phonon DOS.
       - Use QE matdyn.x with dos=.true. on a dense q-mesh (e.g., 20x20x20 interpolation) to get the phonon DOS.
       - ZPE per atom in meV: integrate (1/2)*hbar*omega * g(omega) d(omega).
       - Compute ZPE correction to formation enthalpy: Delta_ZPE = ZPE(candidate) - sum_i x_i * ZPE(element_i). For H2, ZPE ~ 135 meV/molecule (well-known; verify from our H2 phonon or use literature).
       - Report: if |Delta_ZPE| > 25 meV/atom, flag that ZPE could shift E_hull across the 50 meV/atom threshold. This is an ESTIMATE only -- full SSCHA is deferred to Phase 4.
       - Record "zpe_meV_atom" and "delta_zpe_formation_meV_atom" in the JSON output.

    5. **Record and report.** Save to data/candidates/perovskite_phonons.json:
       ```json
       {
         "KGaH3_10GPa": {
           "min_frequency_cm-1": X,
           "qgrid_convergence": {"4x4x4": X1, "6x6x6": X2, "8x8x8": X3_or_null},
           "qgrid_converged_at": "6x6x6" | "8x8x8",
           "stability_verdict": "stable" | "unstable" | "borderline-SSCHA",
           "imaginary_mode_character": "none" | "H-dominated" | "framework",
           "zpe_meV_atom": Z,
           "delta_zpe_formation_meV_atom": DZ,
           "zpe_hull_shift_flag": true | false,
           "notes": "..."
         },
         ...
       }
       ```

    6. **Generate phonon dispersion figures.** One plot per candidate per pressure. Mark imaginary modes clearly (negative frequency convention). Include acoustic mode check at Gamma (must be ~0).

    **APPLY fp-unstable-tc FILTER:** Candidates with persistent imaginary modes > 5 cm^-1 (after 6x6x6 convergence) where modes are framework-dominated are ELIMINATED. They do NOT advance to Phase 3 regardless of literature Tc predictions.

    **If NO candidates pass both filters** (E_hull < 50 meV/atom AND dynamically stable) at any P <= 10 GPa:
    - Document this as a NEGATIVE RESULT for the perovskite family at our target pressure.
    - Check if any are stable at 50 GPa (they should be, per Du et al.) -- this confirms the methodology works but the pressure target is too low for this family.
    - Flag for the synthesis plan in Plan 02-04: "Perovskite family requires P > 10 GPa."
  </action>
  <verify>
    1. Acoustic modes at Gamma: frequency < 2 cm^-1 for all candidates (ASR enforcement working)
    2. q-grid convergence: frequencies change by < 5 cm^-1 between successive q-grids (4x4x4 vs 6x6x6; and 6x6x6 vs 8x8x8 if triggered). 8x8x8 MUST be run if 4x4x4-to-6x6x6 disagreement > 5 cm^-1.
    3. For candidates matching Du et al. predictions (stable at ~10 GPa): our phonon dispersions should be qualitatively similar (no unexpected imaginary branches)
    4. Stability verdict is unambiguous: "stable" means ALL frequencies > -5 cm^-1 at converged q-grid; "unstable" means persistent imaginary modes > 5 cm^-1 with framework character
    4a. ZPE estimation: delta_zpe_formation computed for all stable candidates; hull shift flagged if |Delta_ZPE| > 25 meV/atom
    5. Phonon dispersion plots show correct number of branches: 3N per unit cell (5 atoms in Pm-3m -> 15 branches, 3 acoustic + 12 optical)
    6. fp-unstable-tc applied: no Tc estimates for unstable structures
  </verify>
  <done>Phonon stability determined for all near-hull perovskite candidates; q-grid convergence verified; stability verdicts recorded; phonon dispersion figures produced</done>
