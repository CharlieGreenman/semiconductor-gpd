       - Include Hg1245 both scenarios (paramagnetic and AF-2OP)

    2. Plane-resolved lambda decomposition:
       The total lambda = sum of contributions from each CuO2 plane:
       lambda_total = sum_i lambda_i where i runs over all CuO2 planes

       For Hg1223 (n=3): lambda = lambda_IP + 2*lambda_OP
       For Hg1234 (n=4): lambda = 2*lambda_IP + 2*lambda_OP
       For Hg1245 paramagnetic (n=5): lambda = 3*lambda_IP + 2*lambda_OP
       For Hg1245 AF (n=5): lambda_eff = 2*lambda_OP (inner planes gapped)

       Estimate lambda_IP and lambda_OP per plane:
       - lambda_OP ~ lambda(Hg1201) / 1 ~ 0.4-0.5 per OP (Hg1201 is single-layer, provides OP baseline)
       - lambda_IP can be extracted: lambda_IP = [lambda_total - n_OP * lambda_OP] / n_IP
       - For Hg1223: lambda_IP = [1.19 - 2*0.45] / 1 ~ 0.29 per IP
       - This means IP has LOWER phonon lambda than OP (less apical O coupling)
       - BUT IP has higher superconducting gap (from NMR/STS) -- the extra gap comes from spin fluctuations

    3. Tc vs n_layers figure (figures/hg_family/tc_vs_nlayers.pdf):
       - x-axis: n_layers (3, 4, 5)
       - y-axis: Tc (K)
       - Plot: Tc_phonon (blue, with mu* shading), Tc_expt (red diamonds), Tc_Hg1245_2OP (green dashed)
       - Annotate the gap between phonon-only and experimental as "spin-fluctuation contribution"
       - Show that Tc_expt peaks at n=3 while Tc_phonon(paramagnetic) increases monotonically

    4. Lambda decomposition figure (figures/hg_family/lambda_decomposition.pdf):
       - Stacked bar chart: IP contribution (dark blue) + OP contribution (light blue) for each compound
       - Show that OP contribution stays roughly constant (~0.9) while IP contribution grows with n
       - For Hg1245 AF scenario: hatched bars showing only OP contribution is active

    5. Inner vs outer lambda trend (figures/hg_family/inner_vs_outer_lambda.pdf):
       - Plot lambda_per_plane vs n_layers for IP and OP separately
       - OP should be roughly constant across the series
       - IP may show slight changes due to internal pressure effects

    6. Key physical insights to document:
       a. OP lambda ~ 0.4-0.5 per plane is roughly constant (structural environment doesn't change much)
       b. IP lambda ~ 0.3 per plane is LOWER (less apical O hybridization -> less phonon coupling)
       c. Adding layers: paramagnetic lambda increases because N(E_F) increases
       d. But IP develops AF order for n >= 4-5, removing those states from the Fermi surface
       e. Net effect: total Tc peaks at n=3 where IP has not yet gone AF
       f. This is the Tachibana et al. 2025 picture confirmed from the phonon side
