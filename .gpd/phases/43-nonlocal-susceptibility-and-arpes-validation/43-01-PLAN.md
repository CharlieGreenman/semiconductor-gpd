# Plan 43-01: Nonlocal Spin Susceptibility and ARPES Validation

## Objective

Extract chi_cluster(q,omega) from the converged DCA self-energy, compute lambda_sf_cluster,
and validate momentum-dependent spectral function against ARPES observables.

## Context

- Phase 42 DCA results: Z_nodal=0.195, Z_antinodal=0.054 (massive pseudogap)
- v9.0 single-site lambda_sf=1.8
- Expected: lambda_sf_cluster ~ 2.5-3.0 from enhanced (pi,pi) peak

## Tasks

### Task 1: Compute chi_cluster(q,omega) and lambda_sf_cluster
type="auto" verify="numerical"

1. Load DCA self-energy Sigma(K,iw_n) from Phase 42
2. Upfold from 4-site cluster to full BZ using DCA coarse-graining
3. Compute chi_cluster(q,iw_n) via RPA with cluster-dressed Green's function
4. Extract peak at Q=(pi,pi): amplitude, width
5. Compute lambda_sf_cluster via Fermi-surface average
6. Compare with single-site lambda_sf=1.8

### Task 2: Spectral function A(k,omega) and ARPES comparison
type="auto" verify="analytical"

1. Compute A(k,omega) from cluster Green's function using Pade continuation
2. Verify Fermi arc structure: sharp node, suppressed antinode
3. Compare antinodal pseudogap with ARPES data (~30-60 meV for Hg1223/Bi2212)
4. Generate figures

## Success Criteria

- chi_cluster(q,omega) has enhanced (pi,pi) peak vs single-site
- lambda_sf_cluster in 2.0-3.5 range
- Fermi arcs visible in A(k,omega)
- Antinodal gap within factor of 2 of ARPES
