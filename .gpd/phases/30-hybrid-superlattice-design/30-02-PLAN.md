  <files>

  <action>
    Convert each CIF structure from Plan 30-01 into a QE pw.x variable-cell relaxation (vc-relax) input file.

    For each candidate:

    1. **Read CIF** from simulations/superlattice/structures/ using pymatgen or ASE
    2. **Assign ONCV pseudopotentials** for each element:
       - Hg: Hg_ONCV_PBEsol-1.2.upf (or equivalent SG15/PseudoDojo)
       - Ba: Ba_ONCV_PBEsol-1.2.upf
       - Ca: Ca_ONCV_PBEsol-1.0.upf
       - Cu: Cu_ONCV_PBEsol-1.0.upf
       - La: La_ONCV_PBEsol-1.2.upf
       - Ni: Ni_ONCV_PBEsol-1.0.upf
       - O: O_ONCV_PBEsol-1.2.upf
       NOTE: Exact PP filenames will need verification against the actual PP library. Use the naming convention from Plan 27-01 (Hg1223 relaxation) as template.

    3. **Set computational parameters:**
       - calculation = 'vc-relax'
       - ecutwfc = 90 Ry (higher than Hg1223's 80 Ry due to mixed chemistry)
       - ecutrho = 360 Ry (4x ecutwfc for NC-PPs)
       - conv_thr = 1.0d-8 (SCF convergence)
       - forc_conv_thr = 1.0d-4 (force convergence in Ry/Bohr)
       - press_conv_thr = 0.1 (pressure convergence in kbar)
       - mixing_beta = 0.3 (conservative for mixed oxide with d-electrons)
       - occupations = 'smearing', smearing = 'mv', degauss = 0.02
       - nspin = 1 (non-magnetic starting point)

    4. **Set k-point grid:** Scale inversely with cell size.
       - Candidate 1 (~20-24 atoms, a ~ 3.9 A, c ~ 13 A): 8x8x4 MP grid
       - Candidate 2 (~50-60 atoms, a ~ 3.9 A, c ~ 36 A): 6x6x2 MP grid
       - Candidate 3 (~35-40 atoms, a ~ 3.9 A, c ~ 30 A): 6x6x2 MP grid

    5. **Cell constraints:** Allow c-axis and internal coordinates to relax. Optionally constrain in-plane a=b to maintain tetragonal symmetry (cell_dofree = '2Dxy' is NOT what we want; use 'all' for full relaxation or 'z' to relax c only while keeping a,b fixed at the coherent epitaxy value).

    6. **Write atomic positions** in crystal (fractional) coordinates matching the CIF.

    Follow the input format established in simulations/hg1223/structure/hg1223_relax.in from Plan 27-01.
  </action>
  <verify>
    1. Each QE input file is syntactically valid (correct namelist structure, ATOMIC_SPECIES, ATOMIC_POSITIONS, CELL_PARAMETERS, K_POINTS)
    2. Atom count in QE input matches CIF structure for each candidate
    3. Pseudopotential files referenced exist in the expected PP directory (or are flagged as needing download)
    4. ecutwfc >= 80 Ry (sufficient for ONCV with O 2p states)
    5. k-grid density is reasonable: N_k * a > 20 A (i.e., at least 5 k-points per A^-1 along each direction)
    6. Conventions match: Ry for energies, Bohr for internal QE units, crystal coordinates for positions

  <name>Task 2: Estimate E_hull for each candidate against competing phases</name>
  <files>

  <action>
    Estimate the thermodynamic stability (E_hull) of each superlattice candidate.

    **Method:** Since we cannot run DFT (no HPC), use a combination of:
    (a) Published DFT formation enthalpies from Materials Project for parent compounds and competing phases
    (b) Literature values for the formation enthalpy of the parent cuprate and nickelate compounds
    (c) A simple mixing enthalpy estimate for the superlattice

    **Step 1: Identify competing phases for each candidate composition.**
    For [HgBa2CuO4]1/[LaNiO2]1 (Candidate 1, composition: HgBa2LaCuNiO7):
    - Binaries: HgO, BaO, La2O3, CuO, NiO, Cu2O
    - Ternaries: BaCuO2, La2CuO4, LaNiO3, BaHgO2, La2NiO4
    - Parent compounds: HgBa2CuO4, LaNiO2
    - Key question: is the superlattice lower in energy than a mixture of parent compounds?

    For [Hg1223]1/[La3Ni2O7]1 (Candidate 2):
    - All of the above plus CaO, Ca2CuO3, HgBa2Ca2Cu3O8
    - Parent compounds: HgBa2Ca2Cu3O8, La3Ni2O7

    For [HgBa2CuO4]1/[La3Ni2O7]1 (Candidate 3):
    - Similar to Candidate 1 but with La3Ni2O7 instead of LaNiO2
    - Parent compounds: HgBa2CuO4, La3Ni2O7

    **Step 2: Estimate formation enthalpies.**
    Use pymatgen's Materials Project API (if available) or hardcode published values:
    - HgBa2CuO4: Delta_H_f ~ -1.2 to -1.5 eV/atom (from published DFT)
    - La3Ni2O7: Delta_H_f ~ -2.0 to -2.3 eV/atom (from published DFT)
    - For the superlattice: E_superlattice ~ E_parent1 + E_parent2 + E_interface
    - Interface energy E_interface ~ +10 to +50 meV/atom (typical for coherent oxide interfaces)

    **Step 3: Compute E_hull.**
    E_hull = E_superlattice - E_hull_reference (convex hull of competing phases)
    The key decomposition reaction is: superlattice -> parent_cuprate + parent_nickelate
    If this reaction is exothermic, E_hull is dominated by the interface penalty.

    **Expected result:** E_hull ~ 20-80 meV/atom, depending on lattice mismatch strain energy and interface chemistry. Candidates with smaller mismatch should have smaller E_hull.

    **Step 4: Assign verdicts.**
    - E_hull < 50 meV/atom: GO (proceed to phonon calculations)
    - 50 < E_hull < 100 meV/atom: CONDITIONAL (may be kinetically stabilizable; proceed with caution)
    - E_hull > 100 meV/atom: NO-GO (thermodynamically unfavorable)

    Be HONEST: the E_hull estimation without actual DFT is an educated guess with large uncertainty (+/- 30 meV/atom). State this uncertainty explicitly. The interface energy is the largest unknown.

    Save results to data/superlattice/stability_assessment.json and data/superlattice/competing_phases.json.
  </action>
  <verify>
    1. Competing phases list includes at least 6 phases per candidate composition (all stable binaries + key ternaries)

    3. E_hull calculation is documented with explicit arithmetic showing the decomposition reaction
    4. Uncertainty estimate (+/- 30 meV/atom) is stated
    5. Verdict correctly applies the 50 meV/atom threshold with stated uncertainty
    6. The key decomposition (superlattice -> parent compounds) is explicitly evaluated
    7. Interface energy estimate is physically reasonable (10-50 meV/atom for oxide heterointerface)
