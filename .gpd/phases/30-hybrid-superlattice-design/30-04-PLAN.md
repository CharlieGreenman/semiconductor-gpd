    **Step 1: Estimate superlattice lambda and omega_log from parent compound data.**

    Use volume-weighted superposition:
    lambda_SL = (V_cuprate * lambda_cuprate + V_nickelate * lambda_nickelate) / V_total

    Parent compound phonon-only lambda values:
    - Hg1223: lambda = 1.19, omega_log = 291 K (from Phase 27 Plan 02)
    - HgBa2CuO4 (Hg1201): lambda ~ 0.8-1.0, omega_log ~ 250-350 K (published DFT; fewer CuO2 planes = lower lambda)
    - LaNiO2: lambda ~ 0.3-0.5, omega_log ~ 300-500 K (published DFT for infinite-layer nickelate -- weak e-ph coupling)
    - La3Ni2O7: lambda ~ 0.5-0.8, omega_log ~ 250-400 K (published DFT for bilayer RP -- stronger e-ph than LaNiO2)

    Superlattice estimates:
    - Candidate 1 [Hg1201/LaNiO2]: lambda_SL ~ 0.5-0.7, omega_log ~ 280-400 K
    - Candidate 3 [Hg1201/La3Ni2O7]: lambda_SL ~ 0.6-0.8, omega_log ~ 260-370 K

    Interface enhancement factor: published oxide heterostructure literature suggests interface phonon modes can enhance lambda by 10-30% near the interface. Apply a conservative 0-20% enhancement range.

    **Step 2: Compute Allen-Dynes Tc for each candidate.**
    Use the same Allen-Dynes code from Phase 27 (analysis/hg1223/allen_dynes_tc.py) adapted for superlattice inputs.
    - Modified AD formula: Tc = (omega_log / 1.2) * f1 * f2 * exp[-1.04*(1+lambda)/(lambda - mu*(1+0.62*lambda))]
    - Report at mu* = 0.10 and mu* = 0.13
    - Apply semi-analytical Eliashberg correction ratio (same as Phase 27 Plan 03): multiply AD by ~1.05-1.10

    - La3Ni2O7 phonon-only Tc: estimate from literature lambda ~ 0.5-0.8 -> Tc ~ 5-20 K
    - LaNiO2 phonon-only Tc: estimate from literature lambda ~ 0.3-0.5 -> Tc ~ 1-5 K
