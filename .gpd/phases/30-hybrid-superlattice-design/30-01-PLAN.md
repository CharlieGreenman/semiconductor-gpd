    Build 3 cuprate-nickelate Ruddlesden-Popper superlattice structures using pymatgen and/or ASE.

    **Parent compound structures (from experimental crystallography):**

    Cuprate blocks:
    - HgBa2CuO4 (Hg1201): P4/mmm, a ~ 3.88 A, c ~ 9.51 A, 1 CuO2 plane
    - HgBa2Ca2Cu3O8+delta (Hg1223): P4/mmm, a ~ 3.85 A, c ~ 15.78 A, 3 CuO2 planes (from Plan 27-01)

    Nickelate blocks:
    - LaNiO2 (infinite-layer): P4/mmm, a ~ 3.96 A, c ~ 3.37 A, 1 NiO2 plane
    - La3Ni2O7 (RP bilayer): I4/mmm (or Amam under pressure), a ~ 3.96 A, c ~ 20.5 A, 2 NiO2 planes

    **Candidate 1: [HgBa2CuO4]1/[LaNiO2]1 (simplest interface)**
    - Stack 1 unit of Hg1201 on 1 unit of LaNiO2
    - Interface: BaO layer contacts LaO layer (rock-salt interface, common in perovskite heterostructures)
    - In-plane lattice: average of a_Hg1201 and a_LaNiO2, or forced to one value (coherent epitaxy)
    - Mismatch estimate: (3.96 - 3.88) / 3.88 ~ 2.1% (within coherent growth range)
    - Expected atom count: ~20-24 atoms per supercell

    **Candidate 2: [Hg1223]1/[La3Ni2O7]1 (multilayer cuprate / bilayer nickelate)**
    - Stack 1 unit of Hg1223 on 1 unit of La3Ni2O7
    - Interface: BaO-LaO rock-salt contact
    - Mismatch: (3.96 - 3.85) / 3.85 ~ 2.9% (borderline but feasible)
    - Expected atom count: ~50-60 atoms (large unit cell)
    - This is the most ambitious but tests whether multilayer cuprate Tc can be maintained next to a nickelate control surface

    **Candidate 3: [HgBa2CuO4]1/[La3Ni2O7]1 (single cuprate / bilayer nickelate, most MBE-realistic)**
    - Stack 1 unit of Hg1201 on 1 unit of La3Ni2O7
    - Interface: BaO-LaO rock-salt
    - Mismatch: same as Candidate 1 (~2.1%)
    - Expected atom count: ~35-40 atoms
    - Most realistic for MBE growth (simpler cuprate block, well-studied nickelate block)

    For each candidate:
    1. Start from experimental parent structures (use pymatgen's Structure class or ASE Atoms)
    2. Build the superlattice by stacking along c-axis with appropriate interface layer
    3. Set in-plane lattice to average of parent a-parameters (coherent epitaxy)
    4. Relax c-axis and internal coordinates conceptually (mark as "needs DFT relaxation")
    5. Export to CIF format with full symmetry annotation
    6. Verify: correct atom count, no overlapping atoms (min interatomic distance > 1.5 A), charge balance

    Save machine-readable summary to data/superlattice/candidate_structures.json with fields:
    candidate_id, formula, space_group, a, c, n_atoms, mismatch_pct, parent_cuprate, parent_nickelate, interface_type

    Use published experimental lattice parameters. Do NOT invent values. Cite sources for each parent structure.
