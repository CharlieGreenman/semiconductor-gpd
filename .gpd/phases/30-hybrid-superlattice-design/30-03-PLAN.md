    For each candidate that passed E_hull screening in Plan 30-02 (expected: at least Candidates 1 and 3 based on lower mismatch), prepare QE electronic structure calculation inputs.

    NOTE: If Plan 30-02 found ALL candidates have E_hull > 100 meV/atom, this plan should prepare inputs only for the most promising candidate (lowest E_hull) as a learning exercise, and document the others as NO-GO.

    For each viable candidate, prepare 3 QE input files following the template from Phase 27 (simulations/hg1223/electronic/):

    **1. SCF calculation (candidate{N}_scf.in):**
    - Start from the relaxation input (Plan 30-02) but change calculation = 'scf'
    - Denser k-grid: multiply Plan 30-02 k-grid by 1.5x in each direction
    - Same pseudopotentials, ecutwfc, ecutrho, smearing
    - This establishes the charge density for subsequent calculations

    **2. Band structure (candidate{N}_bands.in):**
    - calculation = 'bands'
    - High-symmetry k-path for tetragonal BZ: Gamma-X-M-Gamma-Z-R-A-Z (Setyawan-Curtarolo)
    - 20 k-points per segment
    - nbnd = n_electrons/2 + 20 (enough conduction bands to see hybridization)

    **3. NSCF for DOS (candidate{N}_nscf.in):**
    - calculation = 'nscf'
    - Dense uniform k-grid: 12x12x6 for Candidate 1, 10x10x4 for Candidate 3
    - occupations = 'tetrahedra_opt' for accurate DOS
    - nbnd same as bands calculation

    Also prepare projwfc.x input for projected DOS (Cu-d, Ni-d, O-p decomposition).
  </action>
  <verify>
    1. SCF, bands, and NSCF inputs exist for each viable candidate
    2. Band k-path follows Setyawan-Curtarolo convention for tetragonal
    3. NSCF k-grid is denser than SCF (sufficient for smooth DOS)
    4. Pseudopotentials match Plan 30-02 relaxation inputs
    5. All inputs reference the correct structure (matching Plan 30-01 CIF atom counts)
    6. Only candidates that passed E_hull screening have inputs prepared
  </verify>
  <done>QE electronic structure inputs prepared for all viable candidates with SCF, bands, and NSCF calculations, following Phase 27 conventions</done>

  <action>
    Since no HPC is available, estimate the electronic structure of each viable superlattice candidate using parent compound data.

    **Step 1: Collect parent compound electronic data from literature.**

    HgBa2CuO4 (Hg1201):
    - Published DFT band structure: 1 Cu-O antibonding band crossing E_F, plus O-p bands below
    - N(E_F) ~ 1.5-2.0 states/eV/cell (per spin) -- fewer CuO2 planes than Hg1223
    - Cu-d character: ~55%, O-p: ~40% at E_F
    - Bandwidth W ~ 2 eV for the antibonding band

    Hg1223 (from Plan 27-01):
    - N(E_F) = 4.0 states/eV/cell (total both spins) = 2.0 per spin
    - Cu-d 55% + O-p 39% at E_F
    - 3 Cu-O antibonding bands crossing E_F

    LaNiO2 (infinite-layer nickelate):
    - Published DFT: 1 Ni-d_{x2-y2} band crossing E_F + La 5d pocket at Gamma
    - N(E_F) ~ 1.0-1.5 states/eV/cell (per spin)
    - Ni-d character: ~60%, O-p: ~20%, La-d: ~15% at E_F

    La3Ni2O7 (bilayer RP nickelate):
    - Published DFT: 2 Ni-d_{x2-y2} bands + Ni-d_{z2} band crossing E_F
    - N(E_F) ~ 3.0-4.0 states/eV/cell (per spin) -- bilayer splitting
    - Ni-d character: ~50%, O-p: ~30% at E_F

    **Step 2: Estimate superlattice electronic structure by superposition.**
    For each candidate: N(E_F, superlattice) ~ N(E_F, cuprate block) + N(E_F, nickelate block)
    - Candidate 1: ~ 1.5 + 1.2 = 2.7 states/eV/cell (per spin)
    - Candidate 3: ~ 1.5 + 3.5 = 5.0 states/eV/cell (per spin)
    This is a rough estimate; interface hybridization could increase or decrease N(E_F) by ~20%.

    **Step 3: Produce schematic band structure plots.**
    Using matplotlib, create schematic band structures for each viable candidate showing:
    - Cu-d/O-p bands (from cuprate block, red)
    - Ni-d/O-p bands (from nickelate block, blue)
    - Interface hybridization zones (where Cu-d and Ni-d bands would interact, green dashed)
    - Fermi level clearly marked
    Label these as "SCHEMATIC -- literature-based estimate, not computed"

    **Step 4: Produce projected DOS comparison figure.**
    Create a 3-panel figure:
    - Panel (a): Parent cuprate DOS (from literature)
    - Panel (b): Parent nickelate DOS (from literature)
    - Panel (c): Estimated superlattice DOS (sum of parents, shifted to common Fermi level)
    Mark the interface hybridization region with uncertainty shading.

    **Step 5: Save electronic_summary.json** with fields per candidate:
    candidate_id, N_EF_per_spin (estimated), Cu_d_fraction, Ni_d_fraction, O_p_fraction,
    n_bands_at_EF (estimated), metallic (boolean), bandwidth_eV, source (literature refs)
