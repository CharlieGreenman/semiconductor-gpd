# Phase 50: CTQMC-Corrected Tc for Hg1223 Variants

## Objective

Recompute Tc predictions for all Hg1223 variants using CTQMC lambda_sf,
determining whether the 242 K prediction survives at 300 K (room temperature).

## Context

- v10.0 best: 242 K [200, 300] for Hg1223 strained+15 GPa (Hubbard-I)
- Phase 49: lambda_sf_CTQMC = 1.916 +/- 0.146 (down 33.5% from HI 2.88)
- Tc depends non-linearly on lambda_total through Allen-Dynes formula
- lambda_total = lambda_ph + lambda_sf; phonon part unchanged

## Tasks

### Task 1: Tc Recomputation for All Variants
type="auto" verify="numerical"

For each Hg1223 variant, recompute Tc using CTQMC lambda_sf:
- Scale each candidate's lambda_sf by the CTQMC/HI ratio (0.665)
- Recompute lambda_total, omega_log_eff, Allen-Dynes Tc
- Apply d-wave Coulomb evasion (mu*=0) and Eliashberg correction (1.12x)
- Propagate both statistical and systematic uncertainties

Done: Full candidate table with CTQMC Tc values.

### Task 2: Uncertainty Tightening Assessment
type="auto"

Compare v10.0 and v11.0 uncertainty brackets:
- v10.0 bracket: [200, 300] K (from lambda_sf range [2.52, 3.60])
- v11.0: propagate [1.62, 2.21] through Eliashberg
- Is the bracket tighter? By how much?

Done: Side-by-side comparison showing bracket tightening.

### Task 3: 300 K Decision
type="auto"

Explicit assessment: does Tc_best reach 300 K within the tightened bracket?
- Yes: room-temperature candidate identified
- No: quantify the gap, explain what physics is missing

Done: Clear Yes/No with confidence and gap accounting.

## Success Criteria

1. Tc recomputed for all variants with CTQMC lambda_sf
2. Combined uncertainty: CTQMC stat + systematic + 10% v_F
3. Bracket tighter than v10.0 [200, 300] K
4. Explicit 300 K assessment
5. If Tc_best < 250 K, overestimate hypothesis confirmed
