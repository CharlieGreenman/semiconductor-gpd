<objective>
Validate the DFT+DFPT+Eliashberg computational pipeline by reproducing the superconducting critical temperature of H3S (Im-3m) at 150 GPa.

Purpose: H3S is the primary benchmark (ref-h3s). Reproducing Tc within 15% of the experimental 203 K establishes that the pipeline (pseudopotentials, convergence parameters, Wannier interpolation, Eliashberg solver) is trustworthy for novel hydride predictions. This is the FIRST calculation in the project -- all subsequent work depends on this validation passing.

Output: QE input files for H3S at 150 GPa (vc-relax, SCF, NSCF, DFPT, EPW); benchmark data (Tc, lambda, omega_log, alpha2F, phonon dispersion, EOS); comparison figures; convergence documentation.
</objective>

    Set up the H3S Im-3m crystal structure and generate all QE input files for the DFT+DFPT pipeline at 150 GPa.

    1. **Structure setup:** Build Im-3m H3S (space group #229, BCC lattice). The conventional cell has 4 formula units (4 S + 12 H = 16 atoms); use the PRIMITIVE cell (1 S + 3 H = 4 atoms). Wyckoff positions: S at (0,0,0), H at (0, 1/2, 1/2) and symmetry equivalents. Initial lattice parameter estimate: a ~ 3.0 A (will be refined by relaxation).

    CRITICAL: Use the PRIMITIVE BCC cell for Im-3m, not the conventional cubic cell. The primitive cell has lattice vectors a1 = a/2*(-1,1,1), a2 = a/2*(1,-1,1), a3 = a/2*(1,1,-1) with 4 atoms.

    2. **Generate vc-relax input** (simulations/h3s/h3s_relax.in):
       - calculation = 'vc-relax', press = 1500.0 (kbar = 150 GPa)
       - ecutwfc = 100 Ry (will convergence-test), ecutrho = 400 Ry
       - smearing = 'mp', degauss = 0.02 Ry
       - k-grid: 24x24x24 Monkhorst-Pack Gamma-centered
       - conv_thr = 1.0d-10, forc_conv_thr = 1.0d-4, press_conv_thr = 0.5 kbar
       - Use ONCV PseudoDojo PBEsol norm-conserving pseudopotentials

    IMPORTANT: Verify after relaxation that the structure remains cubic Im-3m. The 150 GPa pressure is near the R3m-to-Im-3m boundary. If the structure distorts, FALL BACK to 'relax' (fixed cell shape) at 155 GPa instead. Do NOT proceed with a non-Im-3m structure.

    3. **Generate convergence test script** (simulations/h3s/convergence_test.py):
       - Test ecutwfc: 60, 80, 100, 120 Ry. Total energy must converge to < 1 meV/atom between successive cutoffs.
       - Test k-grid: 12^3, 16^3, 20^3, 24^3. DOS at E_F must be stable to 5%.
       - Output: convergence table with selected parameters and justification.

    4. **Generate SCF input** (simulations/h3s/h3s_scf.in):
       - Use relaxed structure from step 2
       - Same ecutwfc, k-grid from convergence test
       - conv_thr = 1.0d-10 (tight for phonon-quality wavefunctions)

    5. **Generate NSCF input** (simulations/h3s/h3s_nscf.in):
       - calculation = 'nscf' on uniform k-grid for Wannier90
       - Same parameters as SCF

    6. **Generate DFPT phonon input** (simulations/h3s/h3s_ph.in):
       - ldisp = .true., nq1=nq2=nq3=6 (6x6x6 coarse q-grid)
       - tr2_ph = 1.0d-14 (tight for e-ph quality)
       - fildvscf = 'dvscf' (needed for EPW)
       - electron_phonon = 'interpolated'

    7. **Phonon dispersion check:** After DFPT, generate phonon dispersion along Gamma-H-N-Gamma-P high-symmetry path using matdyn.x. Verify:
       - No imaginary frequencies (all omega > -5 cm^-1)
       - H-stretching modes at ~100-200 meV (800-1600 cm^-1)
       - Three acoustic modes approaching zero at Gamma (ASR enforced)
       - Qualitative shape matches published H3S phonon bands (Duan et al. 2014)

    UNIT CONVERSION CHECK: QE outputs phonon frequencies in cm^-1. Convert to meV using 1 meV = 8.0655 cm^-1. Pressures from QE are in kbar; divide by 10 for GPa.

    1. **Wannier90 + EPW setup** (simulations/h3s/h3s_epw.in):
       - nbndsub: select bands crossing E_F plus margin (~8-12 Wannier functions for 4-atom H3S cell)
       - Disentanglement window: dis_froz_min = -5.0 eV, dis_froz_max = 5.0 eV below/above E_F
       - num_iter = 500 (Wannier spread minimization)
       - Coarse grids must match DFPT: nk1=nk2=nk3=6, nq1=nq2=nq3=6
       - Fine interpolation grids: START with 20^3 k, 10^3 q

    2. **Wannier interpolation quality check:**
       - Plot Wannier-interpolated band structure vs DFT bands
       - Agreement must be < 10 meV in the frozen window near E_F
       - Check decay of real-space e-ph matrix elements (exponential, not power-law)
       - If mismatch > 50 meV: adjust disentanglement window and re-Wannierize

    3. **Lambda convergence study:**
       - Compute lambda at EPW fine grids: 20^3/10^3, 30^3/15^3, 40^3/20^3
       - Lambda must converge to < 5% between 30^3 and 40^3 k-grids
       - If not converged at 40^3, extend to 60^3/30^3
       - Record lambda, omega_log, omega_2 at each grid for convergence table
       - Use degaussw = 0.05 eV for Gaussian smearing of delta functions; test 0.025 and 0.1 eV

    4. **Eliashberg Tc calculation (at converged grid):**
       - limag = .true. (imaginary-axis Eliashberg)
       - lpade = .true. (Pade analytic continuation)
       - wscut = 1.5 eV (>= 5x max phonon frequency; hydrides need large cutoff)
       - nsiter = 500, conv_thr_iaxis = 1.0d-4
       - Solve at mu* = 0.10 AND mu* = 0.13
       - Record: Tc(mu*=0.10), Tc(mu*=0.13), Delta(0), Z(0)

    FORBIDDEN: Do NOT adjust mu* to get closer to 203 K. If Tc is off, investigate alpha2F convergence, NOT mu*.

    5. **Allen-Dynes cross-check:**
       - Compute Tc_AD = (f1*f2*omega_log/1.2) * exp[-1.04*(1+lambda)/(lambda - mu*(1+0.62*lambda))]
       - At mu* = 0.13: expect Tc_AD ~ 170-190 K (10-20% below Eliashberg)
       - If Allen-Dynes gives HIGHER Tc than Eliashberg: something is wrong (flag for debugging)

    6. **Independent alpha2F verification:**
       - Parse EPW output alpha2F(omega)
       - Independently integrate: lambda_check = 2 * integral[alpha2F(omega)/omega d(omega)] using NumPy trapezoid
       - Must agree with EPW-reported lambda to < 1%
       - Verify alpha2F(omega) >= 0 for all omega (positivity)
       - Check shape: two-peak structure (low-freq S modes ~20-60 meV, high-freq H modes ~100-200 meV)

    7. **Equation of state (VALD-04):**
       - Compute total energy at 6-8 volumes around the equilibrium volume (V0 +/- 5%, 10%, 15%)
       - Each volume: fixed-cell SCF at the specified volume (use vc-relax structures or explicit volume scaling)
       - Fit 3rd-order Birch-Murnaghan EOS: E(V) = E0 + (9/8)*V0*B0*[(V0/V)^(2/3) - 1]^2 * {1 + (B0' - 4)*[(V0/V)^(2/3) - 1]/2}
       - Extract: V0, B0 (bulk modulus), B0' (pressure derivative)
       - Compare P(V) curve with Einaga et al. 2016 XRD data
       - Volume at 150 GPa must be within 3% of experimental value

    8. **Compute omega_log/E_F ratio** to verify Migdal approximation validity:
       - omega_log from EPW output (in K)
       - E_F from SCF output (in eV, convert to K)
       - Ratio should be < 0.1 for Migdal-Eliashberg to be reliable

    9. **Assemble benchmark data** (data/h3s/benchmark_results.json):

    10. **Generate figures:**

        - P(V) equation of state with experimental points
