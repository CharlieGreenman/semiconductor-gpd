<objective>
Assemble the benchmark comparison table, document converged parameters, validate the equation of state, and issue a go/no-go decision for proceeding to novel hydride predictions.

Purpose: This plan synthesizes the H3S (Plan 01) and LaH10 (Plan 02) benchmark results into the deliverables required by claim-benchmark and deliv-benchmark. It produces the benchmark table, convergence report, and comparison figures that are the primary Phase 1 outputs. It also makes the go/no-go decision: if both benchmarks pass the 15% criterion, the pipeline is validated and Phase 2 can proceed.

Output: Benchmark table (computed vs experimental Tc for H3S and LaH10); convergence report; comparison figures; recommended convergence parameters for Phase 2+; go/no-go assessment.
</objective>

       d) Fine grid (EPW) convergence: lambda vs fine k-grid for both systems.

          - Identify the minimum grid needed for < 5% convergence

       e) Smearing (degaussw) sensitivity: lambda at different smearing values. Recommend value.

       f) Eliashberg solver convergence: wscut sensitivity (if tested).

    2. **Recommended parameters for Phase 2+:**
       Based on convergence tests, establish a FIXED set of computational parameters for all subsequent calculations:

       | Parameter | Recommended Value | Justification |
       |-----------|------------------|---------------|
       | ecutwfc | {from convergence} Ry | Converged to < 1 meV/atom |
       | SCF k-grid | {N}x{N}x{N} | DOS(E_F) stable to 5% |
       | DFPT coarse q-grid | {N}x{N}x{N} | lambda stable to 10% at coarse level |
       | EPW fine k-grid | {N}x{N}x{N} | lambda converged to < 5% |
       | EPW fine q-grid | {N}x{N}x{N} | coupled to k-grid |
       | degaussw | {value} eV | lambda stable to 5% |
       | wscut | {value} eV | Eliashberg converged |
       | mu* bracket | 0.10, 0.13 | Standard; NEVER tune |

    3. **Systematic error budget for harmonic pipeline:**
       | Error Source | Magnitude | Direction | Can Be Corrected? |
       |-------------|-----------|-----------|-------------------|
       | Harmonic approximation | ~20-30% lambda | Overestimate | Yes (SSCHA in Phase 4) |
       | mu* uncertainty | ~30-60 K Tc | Both directions | No (irreducible) |
       | PBEsol functional | ~1-3% lattice param | -- | Partially (PBE cross-check) |
       | k/q-grid | < 5% lambda | Random | Yes (convergence test) |
       | Isotropic Eliashberg | ~10-20% Tc | Can go either way | Yes (anisotropic Eliashberg) |

    4. **Convergence summary figure** (figures/convergence_summary.pdf):
       - 2x2 panel: (a) ecutwfc convergence, (b) k-grid convergence, (c) lambda vs fine grid, (d) smearing sensitivity
       - Both systems on each panel (different colors/markers)

    5. **GO / NO-GO assessment:**
       - If BOTH benchmarks pass 15%: GO. Proceed to Phase 2 with recommended parameters.
       - If ONE fails 15% but passes 30%: CONDITIONAL GO with documented caveat. Proceed but flag the discrepancy.
       - If EITHER fails 30%: NO-GO. Trigger backtracking: debug pseudopotentials, functional, convergence. Do NOT proceed to novel predictions.
       - If Tc matches only because mu* was adjusted: NO-GO (fp-tuned-mustar violation).

    6. **omega_log/E_F assessment:**
       - Report ratio for both systems.
       - If both < 0.1: Migdal-Eliashberg is reliable.
       - If either > 0.1: Flag as cautionary for strong-coupling candidates in Phase 3.
