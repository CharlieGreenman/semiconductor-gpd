# Phase 84: Plasmon Pairing Interaction and Combined Tc Prediction

## Objective

Compute plasmon-mediated pairing interaction V_pl(q,omega) for the selected
candidates (cuprate c-axis plasmon, n-SrTiO3). Solve combined phonon + plasmon
Eliashberg equations. Test if plasmon boost exceeds the 240 K Eliashberg ceiling.

## Context

- Phase 83: cuprate c-axis plasmon (10-50 meV) and n-SrTiO3 (68-214 meV) selected
- Single-band plasmon is repulsive; need overscreening for net attraction
- Key question: does lambda_pl add to or compete with lambda_ph?

## Tasks

### Task 1: Compute plasmon pairing interaction (type="auto")

Compute V_pl(q,omega) = V_bare * [1/epsilon(q,omega) - 1/epsilon(q,0)]
using RPA dielectric function. Extract lambda_pl from the attractive window.

### Task 2: Combined phonon + plasmon Tc (type="auto")

Solve combined Eliashberg: alpha^2F_total = alpha^2F_ph + alpha^2F_pl.
Compute Tc_combined and compare with Tc_phonon_only.

### Task 3: 300 K assessment and Track B verdict (type="auto")

Quantify Delta_Tc_pl = Tc_combined - Tc_phonon_only.
Test against 240 K ceiling and 300 K target.
Assess competing effect: does plasmon screening reduce lambda_ph?

## Success Criteria

- lambda_pl extracted for both candidates
- Combined Tc computed
- Plasmon boost vs screening competition quantified
- 300 K assessment explicit
