# Phase 82: Vertex Corrections and Non-Adiabatic Tc Prediction

## Objective

Compute first vertex corrections beyond Migdal-Eliashberg for FeSe/STO and
estimate non-adiabatic Tc using Pietronero-Grimaldi formalism. Determine if
non-adiabatic effects can push beyond the 240 K Eliashberg ceiling.

## Context

- Phase 81: FeSe/STO selected (omega_D/E_F = 2.0, 8x Tc enhancement)
- Eliashberg ceiling: 240 +/- 30 K (v14.0)
- First-order vertex formula underpredicts FeSe/STO by 4x

## Tasks

### Task 1: Compute vertex correction for FeSe/STO (type="auto")

Implement Pietronero-Grimaldi vertex function Gamma(q,omega) for:
- Forward scattering (small q): dominant channel for non-adiabatic enhancement
- Backward scattering (q ~ 2k_F): typically suppressive
Compute lambda_eff = lambda_0 * (1 + P1(q_c, omega_D/E_F)) where P1 is the first vertex correction.

### Task 2: Solve vertex-corrected gap equation (type="auto")

Solve the non-adiabatic gap equation to get Tc_NA for FeSe/STO.
Compare with Tc_Eliashberg and Tc_expt.

### Task 3: Extrapolate to 300 K candidates (type="auto")

Apply vertex correction formalism to:
- LaH10 starting from Tc_Eliashberg = 241 K
- Hg1223 starting from Tc = 146 K (with hypothetical H-substrate)
Determine: can non-adiabatic enhancement reach 300 K?

## Success Criteria

- Vertex correction sign and magnitude for FeSe/STO
- Tc_NA for FeSe/STO compared with 65 K experimental
- 300 K assessment: does non-adiabatic mechanism reach 300 K?
- Physical bounds on non-adiabatic enhancement
