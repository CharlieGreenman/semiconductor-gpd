# Plan 44-01: Anisotropic Eliashberg Solver with d-Wave Gap for Hg1223

## Objective

Solve momentum-dependent Eliashberg equations on the Hg1223 Fermi surface with combined phonon + spin-fluctuation kernel. Quantify Tc uplift from d-wave gap anisotropy over the isotropic baseline (108 K from Phase 37).

## Context

- @data/hg1223/eliashberg_combined_results.json (isotropic Tc=108 K, lambda_total=2.99)
- @data/hg1223/spin_susceptibility/pairing_results.json (V_d = -0.80 eV, d-wave channel)
- @data/hg1223/epw_results.json (alpha2F, lambda_ph=1.19, omega_log=25.1 meV)

## Tasks

### Task 1: Fermi Surface Discretization and Kernel Construction
type="auto" verify="numerical"

Build a 2D Fermi surface mesh for the CuO2 plane (tetragonal, single-band effective model).
- Parameterize the FS by angle phi (0 to 2pi) with N_k points on the irreducible wedge
- Construct the anisotropic e-ph coupling g(k,k') from alpha2F with FS harmonic decomposition
- Construct the spin-fluctuation kernel V_sf(k,k') peaked at Q=(pi,pi) using RPA susceptibility form
- Build the combined kernel K(k,k') = K_ph(k,k') + K_sf(k,k')
- Minimum 128 k-points on the full FS (32 on irreducible wedge for C4v)

Done when: Combined kernel K(k,k') is constructed and passes symmetry checks (C4v), sum rule reproduces lambda_total=2.99 within 5%.

### Task 2: Anisotropic Eliashberg Equation Solver
type="auto" verify="numerical"

Implement the linearized anisotropic Eliashberg equations on the Matsubara axis:
- Z(k,iwn) = 1 + (pi*T/wn) * sum_{k',n'} K(k,k') * wn' / sqrt(wn'^2 + Delta(k')^2)
- Delta(k,iwn) * Z(k,iwn) = pi*T * sum_{k',n'} [K(k,k') - mu*(k,k')] * Delta(k',iwn') / sqrt(wn'^2 + Delta(k')^2)
- Use the linearized form near Tc: eigenvalue problem lambda_max = 1 determines Tc
- Enforce d-wave symmetry: Delta(k) = Delta_0 * cos(2*phi_k)
- Iterate with Matsubara cutoff omega_c = 10 * omega_max (convergence check)

Done when: Solver converges to lambda_max=1 at a temperature T=Tc_aniso, with d-wave eigenfunction.

### Task 3: Convergence Study and Tc Determination
type="auto" verify="numerical"

Sweep mesh density and verify Tc convergence:
- N_k = 64, 128, 256, 512 points on full FS
- N_matsubara = 256, 512, 1024 frequencies
- Verify Tc stable to +/-2 K on mesh doubling
- Compute Delta(k)/Delta_max profile and verify d-wave nodal structure
- Compute Tc for mu* = 0.10 and mu* = 0.13 bracket

Done when: Tc_aniso converged to +/-2 K, d-wave nodes confirmed along (pi,pi) diagonal.

### Task 4: Anisotropy Ratio and Literature Comparison
type="auto" verify="analytical"

Quantify the anisotropic enhancement:
- Compute Delta_max / Delta_avg (antinodal-to-average gap ratio)
- Compute Tc_aniso / Tc_iso (should be 1.10 - 1.30 for optimally doped cuprates)
- Compare with published anisotropic Eliashberg results for YBCO, Bi2212, Hg1223
- Compute the "anisotropy boost" = (Tc_aniso - Tc_iso) / Tc_iso * 100%
- Verify: Tc_aniso >= 108 K (must exceed isotropic baseline)
- Expected: Tc_aniso in 119-140 K range

Done when: Anisotropy ratio and Tc uplift documented with literature comparison. Results written to JSON.

### Task 5: Results Summary and Phase 45 Readiness
type="auto"

Write results JSON, generate gap function figure, and prepare inputs for Phase 45:
- Save anisotropic_eliashberg_results.json with all key quantities
- Generate Delta(k) polar plot showing d-wave gap structure
- Generate Tc convergence plot vs mesh density
- Document the anisotropic kernel for downstream use in Phase 45

Done when: All artifacts written, figures generated, results ready for Phase 45 combination with cluster DMFT.

## Success Criteria

1. d-wave Delta(k) with correct nodes along (0,0)-(pi,pi)
2. Anisotropic Tc >= 108 K (isotropic baseline); expect 119-140 K
3. Converged to +/-2 K on mesh doubling
4. Delta_max / Delta_avg quantified and compared with literature
