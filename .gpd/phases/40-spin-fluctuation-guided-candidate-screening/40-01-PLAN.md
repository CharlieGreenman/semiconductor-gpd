# Plan 40-01: Spin-Fluctuation-Guided Candidate Screening

## Objective

Screen 5 structural/chemical modifications of Hg1223 (and one nickelate variant) to maximize lambda_sf + lambda_ph and predict Tc. Identify any candidate exceeding 200 K. Assess stability and synthesis route for the top candidate.

## Context

- @data/hg1223/eliashberg_combined_results.json (Tc=108 K baseline, lambda_total=2.99)
- @data/hg1223/dmft/dmft_results.json (Z=0.33, U=3.5 eV)
- @data/hg1223/spin_susceptibility/pairing_results.json (lambda_sf=1.8, V_d=-0.80 eV)
- @data/nickelate/combined_tc_results.json (nickelate Tc=54 K central)
- @scripts/hg1223/eliashberg_combined.py (validated Tc framework)

## Tasks

### Task 1: Build candidate parameter table
type="auto"
Build physics-motivated parameter modifications for 5 candidates using literature-grounded scaling relations. For each: how does the modification change N(E_F), chi(pi,pi), omega_sf, lambda_ph?

done: Parameter table with 5 candidates and scaling rationale

### Task 2: Compute Tc for all candidates
type="auto"
Use the Phase 37 Eliashberg framework (Allen-Dynes modified + Eliashberg correction) with each candidate's modified parameters. Sweep mu* = 0.10-0.13 and lambda_sf uncertainty.

done: Tc predictions with full uncertainty brackets for all candidates

## Success Criteria

1. >= 3 structural modifications screened with full Tc predictions (GD-01)
2. Top 2-3 candidates identified with Tc + error bars (GD-02)
