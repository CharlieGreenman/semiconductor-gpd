## Tasks

### Task 1: Compute effective omega_log for combined kernel
type="auto" verify="analytical"

Compute omega_log_eff using the two-channel formula:
  omega_log_eff = exp[ (lambda_ph * ln(omega_log_ph) + lambda_sf * ln(omega_sf)) / lambda_total ]

Parameters:
- lambda_ph = 1.1927, omega_log_ph = 291.3 K
- lambda_sf = 1.8 +/- 0.6, omega_sf = 476 K (= 41 meV / k_B)
- lambda_total = lambda_ph + lambda_sf

Verify: omega_log_eff should be between omega_log_ph (291 K) and omega_sf (476 K).

### Task 2: Modified Allen-Dynes with combined lambda_total
type="auto" verify="numerical"

Compute Tc using modified Allen-Dynes formula:
  Tc = (omega_log_eff / 1.2) * exp[ -1.04*(1+lambda) / (lambda - mu*(1+0.62*lambda)) ] * f1 * f2

For mu* = 0.10 and mu* = 0.13 (bracket, do NOT tune).

Strong-coupling corrections f1, f2 from Allen-Dynes (1975).

Verify: Tc should be significantly higher than phonon-only 31 K.

### Task 3: Semi-analytical Eliashberg with two-channel kernel
type="auto" verify="numerical"

Implement semi-analytical Eliashberg solution following Marsiglio & Carbotte (2008) approach:
- Phonon channel: use alpha2F from epw_results.json
- Spin-fluctuation channel: model as peaked spectral function at omega_sf = 41 meV
- Combined: alpha2F_total = alpha2F_ph + alpha2F_sf
- Solve for Tc self-consistently

Cross-check: Eliashberg Tc should be ~10% above Allen-Dynes for lambda ~ 3 (strong-coupling regime).

### Task 4: Uncertainty bracket from lambda_sf range
type="auto" verify="numerical"

Sweep lambda_sf over [1.2, 1.8, 2.4] (central +/- 0.6):
- For each lambda_sf: compute lambda_total, omega_log_eff, Tc (Allen-Dynes + Eliashberg correction)
- For each: mu* = [0.10, 0.13]
- Produce Tc bracket: [Tc_low, Tc_central, Tc_high]

Additional uncertainty from v_F deviation (Phase 36): fold in +/- 10% systematic.

### Task 5: Comparison with 151 K benchmark and Track C decision
type="auto" verify="analytical"

- State Tc prediction with full uncertainty bracket
- Compare with 151 K: compute relative error
- DM-04 test: is Tc within 30% of 151 K? (106-196 K window)
- VALD-02: state the 149 K room-temperature gap EXPLICITLY
- Track C decision: UNLOCK or CANCEL based on DM-04 result
- Generate comparison figure

## Verification

- Modified Allen-Dynes reproduces phonon-only Tc = 31 K when lambda_sf = 0 (regression check)
- Eliashberg correction is positive (Tc_Eliashberg > Tc_Allen_Dynes for lambda > 1)
- omega_log_eff is between omega_log_ph and omega_sf (weighted average)
- All Tc values are positive and less than omega_log_eff (physical bound)
- Phonon fraction = lambda_ph / lambda_total is in 20-45% range (v8.0 mechanism analysis)
