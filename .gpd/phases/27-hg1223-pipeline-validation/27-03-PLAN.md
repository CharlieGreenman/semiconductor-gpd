  <action>
    Compute Tc for Hg1223 using isotropic Eliashberg equations and Allen-Dynes formula.

    1. Eliashberg solver (eliashberg_tc.py):
       - Load alpha2F(omega) from data/hg1223/epw_results.json (Plan 27-02 output)
       - Solve the isotropic Eliashberg equations on the Matsubara axis:
         Z(i*omega_n) = 1 + (pi*T/omega_n) * sum_m lambda(n-m) * omega_m / sqrt(omega_m^2 + Delta_m^2)
         Delta(i*omega_n) * Z(i*omega_n) = pi*T * sum_m [lambda(n-m) - mu*(omega_c)] * Delta_m / sqrt(omega_m^2 + Delta_m^2)
         where lambda(n-m) = 2 * integral [alpha2F(omega) * omega / (omega^2 + (omega_n - omega_m)^2)] d(omega)
       - Matsubara cutoff: omega_c = 10 * max(omega_phonon) (sufficient for convergence)
       - Temperature scan: start at T = 300 K, decrease in steps of 5 K (coarse), then 1 K near Tc
       - Tc determined as the temperature where the maximum gap Delta(i*omega_0) -> 0 (within tolerance 1e-4)
       - Compute for mu* = 0.08, 0.10, 0.13, 0.15

       NOTE: If EPW already solved Eliashberg internally (some EPW versions do), cross-check with this independent solver. If EPW did not, this is the primary Tc calculation.

       ALTERNATIVE: If implementing the full Eliashberg solver from scratch is too complex, use EPW's built-in Eliashberg solver (if available) or the simplified McMillan/Allen-Dynes formula with the iterative Eliashberg correction. Document which approach was used.

    2. Allen-Dynes cross-check (allen_dynes_tc.py):
       - Implement: Tc_AD = (f1 * f2 * omega_log / 1.2) * exp[-1.04*(1+lambda) / (lambda - mu*(1+0.62*lambda))]
       - Strong-coupling corrections (Allen & Dynes 1975):
         f1 = [1 + (lambda / Lambda_1)^(3/2)]^(1/3)
         f2 = 1 + (omega_2/omega_log - 1) * lambda^2 / (lambda^2 + Lambda_2^2)
         Lambda_1 = 2.46*(1 + 3.8*mu*), Lambda_2 = 1.82*(1 + 6.3*mu*)*(omega_2/omega_log)
       - omega_2 = sqrt(2/lambda * integral[alpha2F(omega) * omega] d(omega))
       - Compute for same mu* values as Eliashberg

    3. Sensitivity analysis:
       - Plot Tc vs mu* for both Eliashberg and Allen-Dynes (4 mu* values + interpolation)
       - Mark the 128-174 K acceptance window (151 K +/- 15%) on the plot
       - Mark the 106-196 K backtracking window (151 K +/- 30%) on the plot
       - Mark the experimental Tc = 151 K reference line

    4. Save results to data/hg1223/tc_results.json:
       {
         "structure": "HgBa2Ca2Cu3O8+delta",
         "space_group": "P4/mmm",
         "pressure_GPa": 0,
         "functional": "PBEsol",
         "pseudopotentials": "ONCV",
         "lambda": value_from_27-02,
         "omega_log_K": value_from_27-02,
         "omega_log_meV": value_from_27-02,
         "omega_2_meV": computed_value,
         "N_EF_per_spin": value_from_27-02,
         "tc_eliashberg": {
           "mu_0.08": Tc_K,
           "mu_0.10": Tc_K,
           "mu_0.13": Tc_K,
           "mu_0.15": Tc_K
         },
         "tc_allen_dynes": {
           "mu_0.08": Tc_K,
           "mu_0.10": Tc_K,
           "mu_0.13": Tc_K,
           "mu_0.15": Tc_K
         },
         "method_primary": "isotropic Eliashberg (Matsubara)",
         "method_crosscheck": "Allen-Dynes with f1*f2 corrections"
       }
       This satisfies VALD-02: every Tc specifies structure, pressure, mu*, lambda, omega_log, method.
