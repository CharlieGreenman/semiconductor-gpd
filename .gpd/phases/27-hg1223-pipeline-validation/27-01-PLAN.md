    Build the Hg1223 (HgBa2Ca2Cu3O8+delta) crystal structure and prepare QE relaxation input.

    1. Crystal structure: HgBa2Ca2Cu3O8+delta, space group P4/mmm (No. 123), tetragonal.
       Experimental lattice parameters (optimally doped, ambient): a ~ 3.85 A, c ~ 15.8 A.
       Atomic basis (approximate fractional coordinates for P4/mmm):
       - Hg at (0, 0, 0)
       - Ba at (0.5, 0.5, z_Ba) with z_Ba ~ 0.18
       - Ca at (0.5, 0.5, z_Ca) with z_Ca ~ 0.35
       - Cu1 (outer plane) at (0, 0, z_Cu1) with z_Cu1 ~ 0.28
       - Cu2 (inner plane) at (0, 0, 0.5)
       - O1 (apical) at (0, 0, z_O1) with z_O1 ~ 0.14
       - O2 (outer CuO2 plane) at (0, 0.5, z_Cu1)
       - O3 (inner CuO2 plane) at (0, 0.5, 0.5)
       - O_delta (overdoped interstitial) -- omit for stoichiometric calculation; note this approximation.

       IMPORTANT: Look up the most precise experimental coordinates from the ICSD or published refinements. The coordinates above are approximate starting points. The executor MUST verify against at least one published structural refinement.

    2. QE vc-relax input (hg1223_relax.in):
       - ibrav = 0 (explicit CELL_PARAMETERS in angstrom)
       - calculation = 'vc-relax'
       - ecutwfc: start at 80 Ry; ecutrho = 4 * ecutwfc (for NC pseudopotentials, ecutrho can be set to the default)
       - k-points: 8x8x4 Monkhorst-Pack (tetragonal; c >> a so fewer k_z points needed)
       - conv_thr = 1.0d-8
       - cell_dofree = 'all' (relax both a and c)
       - PBEsol functional (input_dft = 'pbesol')
       - Pseudopotentials: ONCV from SG15 or PseudoDojo for Hg, Ba, Ca, Cu, O
       - Smearing: Marzari-Vanderbilt (cold smearing), degauss = 0.02 Ry

       NOTE on pseudopotentials: Hg is a heavy element. Check whether scalar-relativistic ONCV is sufficient or whether spin-orbit coupling matters for the Fermi surface. For this initial validation, use scalar-relativistic and note the approximation.

       NOTE on magnetism: Hg1223 at optimal doping is non-magnetic (superconducting ground state). Start with nspin=1. If the DOS at E_F is anomalously low or the band structure shows unexpected gaps, consider nspin=2 or DFT+U as diagnostics.

    3. Write analysis/hg1223/structure_comparison.py:
       - Parse QE relax output for final lattice parameters (a, c, c/a) and atomic positions
       - Compare against experimental values
       - Compute percentage errors: delta_a = |a_calc - a_exp|/a_exp * 100
       - Output results to data/hg1223/relaxed_structure.json with fields:
         {a, c, c_over_a, atomic_positions, a_exp, c_exp, error_a_pct, error_c_pct, space_group, functional, ecutwfc}
       - ASSERT: error_a_pct < 2.0 AND error_c_pct < 2.0

    NOTE: This task produces QE INPUT FILES and analysis scripts. The actual QE calculation requires HPC resources and will be noted as a researcher-action dependency. The scripts must be ready to run.

         (Gamma = (0,0,0), X = (0.5,0,0), M = (0.5,0.5,0), Z = (0,0,0.5), R = (0.5,0,0.5), A = (0.5,0.5,0.5))

       - NSCF with dense k-grid: 16x16x8 Monkhorst-Pack

       - Extract N(E_F) in states/eV/cell (both spins) -- REMEMBER: QE dos.x gives total (both spins)

       - Identify bands crossing E_F: count, character, dispersion
       - Check: is the system metallic? N(E_F) > 0?
       - Check: do Cu-d and O-p dominate at E_F?

    Reference: Cuprate electronic structure should show:
