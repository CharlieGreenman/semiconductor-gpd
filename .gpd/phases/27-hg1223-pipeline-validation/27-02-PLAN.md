  <action>
    Compute the phonon dispersion of Hg1223 using DFPT (density-functional perturbation theory).

    1. DFPT phonon calculation (hg1223_ph.in):
       - Start from converged SCF (from Plan 27-01)
       - q-point grid: 4x4x2 (coarse grid for DFPT; Hg1223 has a large unit cell so 4x4x2 may be sufficient; if resources allow, try 6x6x3)
       - tr2_ph = 1.0d-14 (phonon self-consistency threshold)
       - alpha_mix = 0.7
       - epsil = .true. (compute dielectric tensor for LO-TO splitting at Gamma)
       - All irreducible representations
       - Output: dynamical matrices at each q-point

    2. Fourier interpolation:
       - q2r.x (hg1223_q2r.in): interatomic force constants from dynamical matrices
         asr = 'crystal' (enforce acoustic sum rule)
       - matdyn.x (hg1223_matdyn.in): interpolate phonon frequencies along high-symmetry path
         Path: Gamma-X-M-Gamma-Z-R-A-Z (same path as electronic bands for comparison)
         asr = 'crystal'
         Dense path: 100 points per segment

    3. Phonon analysis script (phonon_analysis.py):
       - Parse matdyn output for frequencies along the path
       - Plot phonon dispersion: frequency (cm^-1 or meV) vs q-path
       - Annotate: acoustic branches (3), optical branches (3*N_atoms - 3)
       - For Hg1223 with ~13 atoms/cell: 39 branches total, 3 acoustic + 36 optical
       - Color-code by mode character if projwfc data available (Cu-O modes vs Ba/Ca/Hg modes)
       - STABILITY CHECK: find minimum frequency across entire dispersion
         If min_freq < -5 cm^-1: FLAG as dynamically unstable
         If min_freq > -5 cm^-1 but < 0: note as numerical noise, acceptable
       - Save to data/hg1223/phonon_results.json:
         {min_frequency_cm-1, n_imaginary_modes, q_point_of_min, max_frequency_cm-1,
          acoustic_branches_at_gamma: [should be ~0], stability_verdict: "stable"|"unstable"}

    EXPECTED PHYSICS for Hg1223:
    - Low-frequency modes: Ba, Ca, Hg vibrations (~50-200 cm^-1 / 6-25 meV)
    - Mid-frequency modes: Cu-O bond bending (~200-400 cm^-1 / 25-50 meV)
    - High-frequency modes: Cu-O bond stretching, apical O modes (~400-700 cm^-1 / 50-87 meV)
    - No modes above ~700 cm^-1 (unlike hydrides with H modes at ~1000+ cm^-1)
    - Maximum phonon energy is much lower than in H3S/LaH10 -- this fundamentally limits omega_log and hence Tc via Eliashberg
  </action>
  <verify>
    1. Acoustic sum rule: three branches approach 0 at Gamma (residual < 5 cm^-1 after ASR)
    2. No imaginary modes above -5 cm^-1 threshold
    3. Maximum frequency physically reasonable: 400-700 cm^-1 for a cuprate (no spurious high modes)
    4. Correct number of branches: 3 * N_atoms (13 atoms -> 39 branches)
    5. LO-TO splitting visible at Gamma for polar optical modes
    6. Phonon DOS integrates to 3*N_atoms per cell (normalization check)
  </verify>
  <done>Phonon dispersion computed and plotted; dynamic stability verified (no imaginary modes above -5 cm^-1); phonon frequency range physically reasonable for a cuprate; results saved to JSON with stability verdict</done>

  <action>
    Compute the electron-phonon coupling for Hg1223 using EPW Wannier interpolation.

    1. EPW input (hg1223_epw.in):
       - Coarse grids (from DFPT): nk1,nk2,nk3 = 8,8,4 (electrons); nq1,nq2,nq3 = 4,4,2 (phonons)
       - Fine grids: start with nkf1,nkf2,nkf3 = 20,20,10; nqf1,nqf2,nqf3 = 20,20,10
       - Wannier functions: use proj(Cu) = d, proj(O) = p as initial projections
         This should capture the Cu-d / O-p manifold that dominates the Fermi surface
       - Disentanglement: dis_win_min, dis_win_max, dis_froz_min, dis_froz_max
         Set frozen window around E_F to capture all bands crossing the Fermi level
         Typical: dis_froz_min = E_F - 2 eV, dis_froz_max = E_F + 2 eV
         Outer window: dis_win_min = E_F - 6 eV, dis_win_max = E_F + 6 eV
       - epbwrite = .true., epbread = .false. (first run)
       - ephwrite = .true. (write e-ph matrix elements)
       - Output: alpha2F(omega), lambda, omega_log, omega_2

    2. Convergence test:
       - Run EPW at TWO fine grid densities:
         Grid A: 20x20x10 (k) x 20x20x10 (q)
         Grid B: 30x30x15 (k) x 30x30x15 (q) (if resources allow; otherwise 24x24x12)
       - Compare lambda, omega_log between grids
       - Convergence criterion: |lambda_A - lambda_B| / lambda_B < 0.001 (3 sig figs)

    3. EPW convergence analysis (epw_convergence.py):
       - Parse EPW output for lambda, omega_log at each grid
       - Compute convergence metrics
       - Plot alpha2F(omega) from the converged calculation
       - Plot cumulative lambda(omega) = 2 * integral_0^omega [alpha2F(w)/w] dw
       - Save to data/hg1223/epw_results.json:
         {lambda, omega_log_K, omega_log_meV, omega_2_meV, N_EF_per_spin,
          fine_k_grid, fine_q_grid, n_wannier_functions, wannier_spread_A2,
          alpha2F_data: {omega_meV: [...], alpha2F: [...]}}
       - Save convergence data to data/hg1223/epw_convergence.json:
         {grids: [{grid_label, nkf, nqf, lambda, omega_log}], converged: true/false}

    4. Verify alpha2F sum rule:
       - Compute lambda_integral = 2 * integral[alpha2F(omega)/omega d(omega)]
       - Compare to EPW-reported lambda
       - Must agree within 1%

    EXPECTED PHYSICS for Hg1223:
    - lambda ~ 1.0-2.5 (moderate to strong coupling for a cuprate)
      NOTE: Literature values for cuprate lambda from e-ph alone are often 0.5-1.5;
      spin fluctuations contribute additional pairing. If lambda < 0.5, the phonon-mediated
      channel alone may be too weak to explain Tc ~ 151 K, which is a DIAGNOSTIC FINDING.
    - omega_log ~ 200-500 K (limited by the lower phonon energy scale compared to hydrides)
    - alpha2F should show peaks at Cu-O stretching/bending frequencies (~30-80 meV)
    - Compared to H3S (omega_log ~ 1200 K), the much lower omega_log for cuprates means
      Tc ~ 150 K requires lambda ~ 1.5-2.5 to compensate

    IMPORTANT DIAGNOSTIC: If lambda comes out < 0.5, this means phonon-mediated pairing alone
    cannot explain Tc ~ 151 K. This is NOT a pipeline failure -- it is a physically meaningful
    finding that Hg1223 likely requires beyond-Eliashberg pairing (spin fluctuations). Document
    this as a finding and note that the Eliashberg Tc will systematically underestimate the true Tc.
  </action>
  <verify>
    1. Wannier interpolation: band structure from Wannier matches DFT bands within 10 meV near E_F
    2. Wannier spread is converged (change < 1% between last two iterations)
    3. lambda convergence: 3 significant figures between two grids
    4. alpha2F(omega) >= 0 for all omega > 0
    5. alpha2F sum rule: integral matches EPW lambda within 1%
    6. omega_log physically reasonable: 200-500 K for a cuprate (not hydride-scale ~1000 K)
    7. Migdal check: omega_log / E_F < 0.1 (verify adiabatic approximation)
    8. N(E_F) from EPW matches Plan 27-01 DOS value within 10%
  </verify>
