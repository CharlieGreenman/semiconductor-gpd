# Phase 54: Vertex Corrections for d-Wave Channel

## Objective

Estimate leading vertex correction magnitude and sign for d-wave pairing in Hg1223.
Uses literature estimates calibrated to CTQMC susceptibility from Phase 49.

## Context

- Particle-particle ladder vertex in d-wave: typically +5-15% Tc enhancement (literature)
- But particle-hole channel can suppress Tc if dominant
- Literature consensus for optimally-doped cuprates: net +5-10% correction
- This is the last theoretical handle before the final assessment

## Tasks

1. **Task 1** (type="auto"): Estimate pp-ladder vertex correction in d-wave channel
   - Use Moriya-lambda correction + DGA estimates from literature
   - Calibrate to Hg1223 parameters (U/t, doping, T)
   - Done: Vertex correction magnitude and sign determined

2. **Task 2** (type="auto"): Apply vertex correction to Tc predictions
   - Modify effective lambda_sf by vertex correction factor
   - Recompute Tc with corrected interaction
   - Done: Vertex-corrected Tc for all variants

3. **Task 3** (type="auto"): Assess significance and document
   - Is |delta_Tc/Tc| > 10%? If not, subdominant
   - Final Tc bracket with vertex corrections included
   - Done: Assessment complete
