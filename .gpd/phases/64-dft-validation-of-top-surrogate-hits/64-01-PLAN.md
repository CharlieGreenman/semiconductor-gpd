# Phase 64: DFT Validation of Top Surrogate Hits

## Objective

Validate the top 10 surrogate hits from Phase 63 with literature-grounded
DFT-level stability estimates, phonon frequency analysis, and Eliashberg
parameter assessment. Determine the false positive rate: how many survive
stability + physics scrutiny?

## Context

- Top 10 hits are all CuO2/LiH superlattices and one LaCuO-H perovskite
- None reach Phase 58 target zone (lambda=2.5-4, omega_log=700-1200 K)
- Best predicted Tc is 82 K (phonon-only, no spin fluctuations)
- No HPC DFT available; use literature-grounded analytical estimates

## Tasks

### Task 1: Stability assessment (E_hull estimates)
type="auto"
- For each top-10 candidate: estimate E_hull from known decomposition products
- Use Materials Project convex hull data for competing phases
- Gate: E_hull < 50 meV/atom required for survival
- Done: stability table for all 10 candidates

### Task 2: Phonon frequency and H-mode analysis
type="auto"  verify="numerical"
- Estimate H-mode frequencies from bond-length scaling and known hydride phonons
- Estimate omega_log from weighted average of oxide and H-mode contributions
- Compare with surrogate omega_log estimates
- Done: phonon analysis for all 10 candidates

### Task 3: Eliashberg parameter estimation
type="auto"  verify="analytical"
- Estimate lambda_ph from N(E_F) and phonon frequency (McMillan-Hopfield)
- Estimate Tc via Allen-Dynes with literature-calibrated parameters
- Compare with surrogate predictions to assess accuracy
- Done: Eliashberg parameters and Tc estimates for surviving candidates

### Task 4: False positive assessment and Phase 58 comparison
type="auto"
- Count candidates surviving stability + physics gates
- Compare validated Tc with surrogate predictions
- Assess whether any validated hit warrants Phase 61/62 pipeline entry
- Done: false positive rate; go/no-go for Track C integration

## Verification

- VALD-02: every candidate assessed for E_hull < 50 meV/atom and phonon stability
- Validated omega_log compared with surrogate estimate
- False positive rate quantified
- No candidate promoted without stability + physics scrutiny
